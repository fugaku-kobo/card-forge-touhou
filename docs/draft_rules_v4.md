# 手塚カードゲーム 仮ルール v4(アナログ版・テストプレイ用)

> ⚠ **これは確定ルールではありません。テストプレイ用の暫定仕様(draft)です。**
> 確定したルールは `docs/rule_status.md` を参照。本ドキュメントは将来 game_design.md へ反映するか検討するための仮置き場です。

最終更新: 2026-05-13(v4.1 ドリームオーダー式バトル + 持ち込みアクション追加)

---

# 1. ゲーム概要

プレイヤーは漫画家。手札のキャラを場に出して育て、**バトル勝利** または **意図的な持ち込み** で原稿を完成させる競争。
バトルは「ドリームオーダー式の伏せ読み合い」で、毎回の戦闘がドラマチック。

## 勝利条件
- **原稿 5 枚先取** で勝利
- **8 ターン終了時** の原稿数比較(同数なら場の魅力合計、それでも同数なら引き分け)

## 敗北条件
- 相手が勝利条件達成
- 自陣全没 + 「泣きの一回」使用済み(レアケース)

# 2. 必要な物品(アナログプレイ前提)

| アイテム | 数 | 用途 |
|---|---|---|
| デッキ(主人公 1 + キャラ + 効果カード) | 各プレイヤー 1 セット | 通常 |
| **ひらめきカウンタ**(d10 ダイス推奨) | 各プレイヤー 1 個 | サイドの IP 残量を物理表示 |
| **原稿ゾーン置き場** | 1 つ | テーブル上のスペース |
| **没ゾーン置き場** | 1 つ | テーブル上のスペース |

→ **進捗トークン・紙の枚数・HP マーカー 等の追加管理は不要**。
   管理するのは「ひらめき」と「ゾーン分け」のみ、アナログでも快適。

---

# 3. セットアップ

1. **先攻決定**: ジャンケンなどでランダムに決定
2. **デッキ準備**: 各プレイヤーがデッキをシャッフル
3. **主人公配置**: 主人公カードを主人公ゾーンに **表向き** で配置(進化なし、ゲーム中ずっと固定)
4. **初期手札**: 7 枚を引く
5. **マリガン**: 初ターンに手札にキャラカードが 1 枚もない場合、公開してシャッフル、7 枚引き直し(回数制限なし、ペナルティなし)

---

# 4. ターン構造(6 フェイズ)

1. **⓪ 開始フェイズ**: ひらめき **+2** 自動加算(カウンタダイスを進める)
2. **① ドローフェイズ**: 2 枚ドロー(先攻 1 ターン目はスキップ)
3. **② アイディアフェイズ**: 手札からキャラを場に出す、効果カード使用、伏せカード設置
4. **③ 持ち込みフェイズ**: バトル宣言(任意)
5. **④ 担当編集フェイズ**: 主人公スキル発動、**持ち込みアクション**、進化、その他効果カード使用
6. **⑤ 就寝フェイズ**: 手札クリンナップ(任意で手札を没へ → IP 獲得)、ターン終了

# 5. キャラクターカード

## 5.1. ステータス

| 項目 | 範囲 | 意味 |
|---|---|---|
| **魅力** | 5-30(カード固有) | バトル時の基礎戦闘力 |
| **奥の手値** | 0-10(カード固有) | 伏せて使った時の戦闘力ブースト |
| **コスト** | 0-5 | 場に出すために必要なひらめき |
| **没で得る IP** | 1-5(逆相関) | 没ゾーン送り時に所有者が獲得するひらめき |
| **Lv** | 1/2/3 | 進化段階(脇役/名脇役/相棒) |

## 5.2. 奥の手値の目安(全カード共通)

各カードに「奥の手値」が印字されている。バトル時に伏せて使うとブースト発生。
強いカードほど奥の手値が高く、没で得る IP が低い(=逆相関、FAB 風)。

| カードタイプ | 奥の手値 | 没で得る IP | 性格 |
|---|---|---|---|
| Lv1 キャラ(脇役) | 1-3 | 4-5 | 切るの軽い |
| Lv2 キャラ(名脇役) | 3-5 | 3 | 中庸 |
| Lv3 キャラ(相棒) | 5-7 | 1-2 | 切るの重い |
| 自信作 | +2 ボーナス | 同 Lv | 強い |
| 展開アイディア | 4-6 | 2-3 | 中庸 |
| テコ入れ | 5-8 | 1-2 | やや重い |
| どんでん返し | 7-10 | 1 | 重い |

## 5.3. 進化(描き込み)

- 同名キャラの **Lv1 → Lv2 → Lv3** で進化
- 進化先カードを進化元の **上に重ねる**(進化スタック保持)
- 進化に必要: ひらめき消費(暫定 2)、 場に出て 1 ターン経過後
- 進化したターンに攻撃可

## 5.4. 場の上限

- 自陣 **4 体まで**
- 上限到達時、新キャラ出せない
- ④ フェイズで自陣のキャラを **任意で没にできる**(ピッチで IP 獲得、場のスペース確保)

# 6. バトル(③ 持ち込みフェイズ・ドリームオーダー式)

## 6.1. 攻撃手順

```
1. 攻撃宣言
   攻撃側: 自陣のキャラ 1 体を指定(ひらめき 1 消費)
            → 相手陣のキャラ 1 体をターゲット指定
            ※ 主人公は対象外、場に出したターンのキャラは攻撃不可

2. 伏せフェイズ(先攻側=攻撃側 → 後攻側=防御側 の順)
   ├ 攻撃側: 手札から 0-2 枚を伏せて置く(任意)
   └ 防御側: 攻撃側の枚数を見て、0-2 枚を伏せて置く(任意)
   ※ 各カード 1 枚伏せるごとに ひらめき 1 消費

3. 一斉オープン
   両者、伏せたカードを同時にめくる

4. 戦闘力計算
   攻撃側戦闘力 = キャラの魅力 + 伏せたカードの奥の手値合計
   防御側戦闘力 = キャラの魅力 + 伏せたカードの奥の手値合計

5. 結果適用
   ┌─ 戦闘力高い方が勝者
   ├─ 敗者のキャラ → 没ゾーン送り(キャラの「没で得る IP」を所有者が獲得)
   ├─ 勝者は自陣のレディキャラ 1 体を原稿化できる(後述)
   └─ 同値 → 両者没(キャラの IP は両者獲得)、原稿化なし

6. 伏せたカードの処理(両者共通)
   伏せたカードは全て没ゾーンへ
   → そのカードの「没で得る IP」を所有者が獲得(ピッチ)
```

## 6.2. 勝利者の権利:自陣キャラの原稿化

**重要**: 攻撃側勝者 / 防御側勝者 のどちらも、勝てば原稿化できる。

- 勝者は自陣のレディキャラから 1 体を選び、**原稿ゾーン** へ送る
- 原稿ゲット数は Lv 別:

| キャラ Lv | 原稿獲得 | 自信作なら |
|---|---|---|
| Lv1 (脇役) | **+0.5**(2 体分で 1 原稿) | +1 |
| Lv2 (名脇役) | **+1** | +2 |
| Lv3 (相棒) | **+2** | +3 |

- 主人公は対象外(主人公は原稿化されない)
- 自陣にキャラがいない場合は原稿化スキップ(原稿は増えない)
- バトルに参加した攻撃キャラ自身も選択可(=持ち込み感)

→ 「**倒したら原稿** ではなく、**勝った権利で自陣のキャラを完成・卒業させる**」
   = 漫画家が「ライバルを退けて自作品を雑誌に通す」演出

## 6.3. 攻撃の制限

- **召喚酔い**: 場に出したターンのキャラは攻撃不可
- 進化したターンは攻撃可
- 先攻 1 ターン目は ③ フェイズスキップ(攻撃不可)
- **1 ターンに 1 攻撃のみ**(濃い読み合いを 1 回に集約)

## 6.4. 戦略の核心

### 攻撃側の判断
- 「強カードを切る = 確実に勝てるが、温存できない」
- 「弱カードを切る = ブラフ気味、IP は稼げる」
- 「0 枚伏せ = 純粋な魅力勝負、リソース節約」
- 「どの自陣キャラを原稿化するか = 育てた愛着のあるキャラ手放す」

### 防御側の判断
- 「先攻 2 枚伏せ → 本気だな、こちらも切るか降参するか」
- 「先攻 0 枚 → ブラフ?余裕?こちらは温存できる」
- 「キャラ犠牲覚悟で強カード温存する選択も」

→ 毎バトルで心理戦、漫画家の「奥の手」演出

# 7. 持ち込みアクション(④ 担当編集フェイズ・バトル無しの原稿化)

## 7.1. 概要

> 攻撃せずに原稿を進めたい場合、「持ち込み」アクションで自陣のキャラを直接編集部へ持ち込める。
> ただし相手の **テコ入れカード** で阻止される可能性あり。

## 7.2. 持ち込み手順

```
1. 持ち込み宣言
   ④ フェイズで、自陣のレディキャラ 1 体(Lv2 or Lv3)を指定
   → ひらめき 3 消費

2. 阻止チャンス
   相手は伏せていた「テコ入れカード」を発動できる(任意)
   ※ テコ入れカードは ② フェイズ等で事前に伏せて置いておく必要あり

3. 結果
   ├─ テコ入れ発動なし → 対象キャラが原稿化(Lv 別の原稿獲得)
   └─ テコ入れ発動    → テコ入れの効果が適用、持ち込み失敗(キャラはそのまま)
                          ※ ひらめき 3 は返ってこない(コスト消費済み)
```

## 7.3. 持ち込み可能なキャラ

| Lv | 持ち込み可? | 原稿獲得 |
|---|---|---|
| Lv1 (脇役) | × 持ち込み不可(まだ未熟) | — |
| Lv2 (名脇役) | ◯ | +1(自信作 +2) |
| Lv3 (相棒) | ◯ | +2(自信作 +3) |

## 7.4. テコ入れによる阻止の例

テコ入れカードの効果例:
- **「企画却下」**: 持ち込みキャラを没ゾーンへ送る + 相手の原稿 -1
- **「打ち切り」**: 持ち込みキャラを没ゾーンへ送る
- **「修正指示」**: 持ち込みキャラはレディ → 横向き(=次ターン使えない)
- **「コンセプト変更」**: 持ち込みキャラを手札に戻す

→ テコ入れの種類によって阻止のキツさが変わる、デッキ構築の楽しみ

## 7.5. 戦略の二重ジレンマ

| プレイヤー | ジレンマ |
|---|---|
| **持ち込み側** | バトル無しの安全策 vs ひらめき消費の重さ |
| **防御側** | テコ入れを「バトルブースト用」に温存 vs 「持ち込み阻止用」に伏せておく |

→ テコ入れカードの使い道が **「バトルの伏せ」「持ち込み阻止」「他効果」** の 3 通り
   = カードの戦略的価値が増す

# 8. 主人公カード

## 8.1. 仕様

- **レベル制廃止**(Lv1/2/3 の段階進化なし)
- 各主人公は 1 枚のカードで完結
- 試合中ずっと主人公ゾーンに居続ける(進化・移動なし)
- バトルの対象にならない(独立、無敵)

## 8.2. 3 つの能力

各主人公カードに以下の 3 種の能力が記載:

| 能力 | 説明 | 例 |
|---|---|---|
| **passive(常在能力)** | 場に居る限り効果発動、コスト不要 | 「自陣に医療キャラがいる間、ターン終了時に手札 +1」 |
| **active(起動能力)** | ひらめきコスト払って 1 ターン 1 回発動 | 「ひらめき 5: 自陣キャラ 1 体を強化」 |
| **awakening(覚醒能力)** | カード裏面に記載、覚醒で起動 | 「ひらめき 3: 強力な効果」 |

## 8.3. 覚醒(awakening)

1. **発動条件**: 相手の原稿が **4 枚以上** に達した瞬間、覚醒可能状態に
   - 5 原稿勝利の 80%、終盤の逆転チャンス
2. **発動方法**: 自分のターン中、好きなタイミングで主人公カードを **物理的に裏返す**
3. 以降、active スキルは裏面の **awakening スキル** に置き換わる
4. 覚醒ボーナス(裏返した瞬間): **自陣キャラ全員の魅力 +5(ターン中)**
5. 一度裏返したら戻らない(永続)

## 8.4. 主人公カード設計例

### ブラック・ジャック(医療系・サポート型)
- passive: 自陣に医療ジャンルキャラが居る間、ターン終了時に手札 +1
- active(コスト 5): 自陣キャラ 1 体の戦闘力 +5(次回バトル時)
- awakening(コスト 3): バトル勝利時、自陣キャラを **2 体** 原稿化できる

### 鉄腕アトム(SF系・加速型)
- passive: 開始フェイズのひらめき +1(=計 +3/ターン)
- active(コスト 4): 自陣 SF キャラ全員の魅力 +5(ターン中)
- awakening(コスト 6): 自陣の SF キャラを **無条件で原稿化**(1 体)

### 火の鳥(歴史/SF・復活型)
- passive: 自陣キャラが没になる時、50% で没ゾーンから手札へ復活
- active(コスト 8): 没ゾーンから 1 体を場に復活
- awakening(コスト 5): 没ゾーンから 2 体を場に復活、自陣全員 +3 魅力

# 9. ひらめき(IP)

## 9.1. 取得

| 手段 | 量 |
|---|---|
| ターン開始時(⓪) | 自動 **+2** |
| カード没時(ピッチ) | カード固定値(1-5) |
| インスピレーション送り | 手札を任意で没へ → そのカードの IP |
| 主人公 passive | カード依存 |
| 効果カード | カード依存 |

## 9.2. 使用

| 用途 | コスト |
|---|---|
| キャラ場出し | カードの「コスト」 |
| バトル攻撃宣言 | 1 |
| バトル伏せカード | 1 枚あたり 1(0-2 枚) |
| 主人公 active | カード依存(3-9) |
| 持ち込みアクション | 3 |
| 効果カード使用 | カード依存 |

## 9.3. 上限

- なし(累積可)
- ダイスで数値を物理表示

# 10. 没ゾーンとピッチシステム

## 10.1. 没ゾーン送り条件

- バトル敗北したキャラ
- バトル中に伏せて使ったカード(全部、勝敗問わず)
- 持ち込み阻止で没にされたキャラ
- 効果カード「ボツ出し」「打ち切り」等で送られたキャラ
- インスピレーション送り(任意)
- ④ フェイズで自主的に自陣キャラを没にする(場の空けと IP獲得)

## 10.2. ピッチシステム(既実装)

- **没ゾーン送りの瞬間**、そのカードの「没で得る IP」分のひらめきを所有者が獲得
- 例外なし(全没でピッチ発動)
- 強いカード(高コスト・高魅力)= 没で得る IP 低い
- 弱いカード(低コスト・低魅力)= 没で得る IP 高い
- → 「弱カードを意図的に没にしてリソース獲得」「強カードを温存」のジレンマ

## 10.3. 没ゾーンの復活(リフレッシュ)

- ドロー時にデッキが空 → 没ゾーンをシャッフルしてデッキへ戻す
- 没ゾーンも空なら、その時点で **敗北**(極めて稀)

# 11. 効果カード(展開アイディア / テコ入れ / どんでん返し)

## 11.1. 共通

- 効果カードは **3 つの使い方** が可能:
  1. **バトル時の伏せカード**(③ フェイズ) — 戦闘力にブースト追加
  2. **持ち込み阻止カード**(④ フェイズ・テコ入れ専用) — 持ち込み宣言時に発動
  3. **通常使用**(② or ④ フェイズ) — カード固有の効果発動

## 11.2. 種類別の役割

### 展開アイディア
- **主に通常使用**(② or ④で効果発動)
- バトル伏せにも使える(汎用ブースト)
- 例: 「ペン入れ集中: 自陣キャラ 1 体の魅力 +5(ターン中)」

### テコ入れ
- **主に持ち込み阻止**(④で相手の持ち込みに対応)
- バトル伏せにも使える(高ブースト)
- 例: 「企画却下: 持ち込みキャラを没へ、相手の原稿 -1」

### どんでん返し
- **特殊条件で発動**(自分の原稿が一定数以上等)
- 強力な逆転効果
- 例: 「大ヒット: 自分の原稿 +2(自分の原稿 2 枚以上で発動可)」

## 11.3. 暫定カード例(調整中、`docs/rule_status.md` の D カテゴリと連動)

| カード | 種類 | コスト | 奥の手値 | 没IP | 効果 |
|---|---|---|---|---|---|
| ペン入れ集中 | 展開 | 3 | 5 | 2 | 自陣キャラ 1 体の魅力 +5(ターン中) |
| アシスタント雇用 | 展開 | 2 | 4 | 3 | 自陣キャラ 1 体の魅力 +3、1 ドロー |
| 企画却下 | テコ入れ | 4 | 6 | 1 | 持ち込みキャラを没へ、相手の原稿 -1 |
| 打ち切り | テコ入れ | 3 | 5 | 2 | 持ち込みキャラを没へ |
| 修正指示 | テコ入れ | 2 | 4 | 3 | 持ち込みキャラはレディ → 横向き |
| 大ヒット | どんでん返し | 5 | 8 | 1 | 自分の原稿 +2(原稿 2 枚以上で発動可) |
| 名作回 | どんでん返し | 6 | 9 | 1 | 自陣の Lv3 キャラを原稿化 |

# 12. 泣きの一回

- **発動条件**: 場のキャラが全没した瞬間(1 試合 1 回限り)
- **発動効果**:
  - 没ゾーンから Lv1 キャラ 1 体を場に復活
  - 自陣全キャラの魅力 +5(ターン中)
- **ペナルティ**: 次の自分ターンは攻撃不可、持ち込み不可

---

# 13. 試合時間の目安

| 段階 | 想定時間 |
|---|---|
| ターン 1-2(序盤、場固め) | 約 1 分/ターン |
| ターン 3-4(中盤、バトル開始) | 約 2 分/ターン |
| ターン 5-6(中盤後半、原稿レース) | 約 2-3 分/ターン |
| ターン 7-8(終盤、覚醒・どんでん返し) | 約 3 分/ターン |
| **1 試合合計** | **約 15-20 分** |

# 14. デッキ構築

## 14.1. デッキ枚数

- メインデッキ: **30-40 枚**(暫定)
- 主人公カード 1 枚必須

## 14.2. カード制限

- 同名カード最大 4 枚
- 主人公は 1 枚のみ
- 一部のカードに「無制限」フラグあり

## 14.3. デッキバランス(推奨、暫定)

- キャラカード: 60-70%(20-28 枚)
  - Lv1 中心 + 進化先 Lv2/Lv3
- 効果カード: 25-30%(8-12 枚)
  - 展開・テコ入れ・どんでん返しのバランス
- 主人公: 1 枚

5. **持ち込みのひらめきコスト**: 3 が適切か、要テスト
6. **奥の手値の数値バランス**: 1-10 の範囲が広すぎないか
7. **同時完成時の処理**: 1 ターンに複数キャラの原稿化(現状: バトル / 持ち込みは 1 ターン 1 回ずつ可)
8. **効果カードの詳細仕様**: D カテゴリ 16 項目要確定
9. **テコ入れの伏せタイミング**: ② フェイズで事前に伏せる必要があるか

# 16. ゲームの楽しさの設計意図

## 1. 「育てたキャラを原稿化する」が中心
- 攻撃 = 直接得点 ではなく、勝利の権利で自陣キャラを完成
- 漫画家の「持ち込み」感を再現

## 2. バトル = ドリームオーダー式の心理戦
- 毎バトル「強カード切るか温存か」のジレンマ
- 先攻 / 後攻の非対称設計で読み合いが深まる

## 3. ピッチで没が意味を持つ
- カードを没にすることでひらめき獲得 → リソース循環
- 強カード(没 IP 低)を切るのは重い決断

## 4. 持ち込みで攻撃しないターンも有意義
- バトル無しの原稿化経路
- テコ入れによる阻止で緊張感

## 5. 覚醒で逆転ドラマ
- ピンチで主人公が物理的に裏返り、強化される

## 6. HP 計算ゼロ、アナログでもサクサク
- 戦闘力は魅力 + 奥の手値の足し算 1 回のみ
- 管理する数値はひらめきのみ

## 7. 試合時間 15-20 分
- 8 ターン制で予測可能、テストプレイ容易

# 18. 次回検討事項(v4.2 候補、本文に未反映)

> ⚠ **以下は 2026-05-14 の議論で出た改善案。本文ルールにはまだ反映していない。**
> 次回再開時にこの章から議論を継続する。

## 18.1. リソース回転の改善(IP過剰 / 手札枯渇 問題)

### 検証結果
- IP は 8 ターンで +48〜58 入る一方、消費は -36〜53 で **+5〜+22 余りがち**
- 手札は ドロー +16〜17 vs 消費 -15〜21 で **-5〜+2 カツカツ**
- → IP余剰と手札不足が同時発生、テンポ悪化

### 採用候補ルール(議論 OK 済み、本文未反映)
1. **「ネタの仕入れ」アクション追加**: ターン中、ひらめき 3 で 1 ドロー(何度でも実行可)
2. **バトル伏せの IP コスト撤廃**: 伏せ自体はひらめき消費なし(手札 -1 のみ)
3. **Lv1 キャラのコスト 0**: 序盤の場展開を加速

→ これで IP↔手札 の交換が成立、リソース管理が良好になる見込み

## 18.2. 技カードの分離(バトル専用カードの新設)

### 現状の問題
- 「奥の手値」を全カード(キャラ・効果カード含む)に持たせる設計だった
- ユーザー指摘:「技カードはバトル時にしか使えないカードであるべき、キャラと差別化したい」
- 「奥の手」というネーミングも違和感

### 採用候補ルール(議論 OK 済み、本文未反映)

#### カード分類の整理(6 種類)
| 種類 | 場に出る? | 通常使用 | バトル伏せ | 用途 |
|---|---|---|---|---|
| 主人公 | 主人公ゾーン | passive/active | × | リーダー |
| キャラ | 場 | × | × | 戦闘員 |
| **技カード**(新) | × | × | ◯ 専用 | バトル強化(強力サポート) |
| 展開アイディア | × | ◯ | × | 一般効果 |
| テコ入れ | 伏せ専用ゾーン | × | ◯(持ち込み阻止) | 阻止 |
| どんでん返し | × | ◯(条件) | × | 逆転 |

#### 数値名の変更
- **「奥の手値」→「見せ場値」** に改名
- 漫画家テーマと完全一致(=バトルの見せ場を作る)

#### バトル伏せのルール
- バトル時に伏せられるのは **技カードのみ**(キャラ・効果カードは伏せ不可)
- 1 バトルあたり伏せ枚数は 1-2 枚(検討中、1 枚案 推奨)
