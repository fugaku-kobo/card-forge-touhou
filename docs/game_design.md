# 手塚カードゲーム 設計ドキュメント

最終更新: 2026-05-13

## このドキュメントについて

手塚カードゲームの **世界観・コンセプト** と **採用検討中の改善提案(L1〜L6)** を 1 つに統合したマスター設計書。
9 ゲームのデモプレイ(`docs/playtest_demos.md`)と TCG 研究(`docs/tcg_research.md`)を経て、本書の提案にたどり着いた。

旧版で 18 個の提案を並べたが「すべて微妙」というユーザー評価を受け、根本から再設計した。
v2 の方針:
- **手塚らしさ(漫画家)を活かす独自設計** を最優先
- **9 ゲームから「真に面白いと検証済みの仕組み」** を選定して統合
- 旧案 F〜I、統合案 J は L1-L6 に吸収・廃止

# 第1部. 基本コンセプト・世界観

## 1.1. 基本コンセプト

> **プレイヤーは漫画家。手札のキャラを育て、名作を生み出す競争をする。**

- ライバル漫画家(対戦相手)に **先に原稿 6 枚を完成** させた方が勝利
- カード = 自分が描くキャラ・アイディア
- ゲームの根幹は **「人気のあるキャラを描き上げる」競争**

## 1.2. 世界観マッピング(用語統一)

### 既に世界観に乗っている用語(現状維持)

| ゲーム用語 | 世界観上の意味 |
|---|---|
| 原稿 | 完成原稿(勝利点) |
| 没ゾーン | ボツ案・打ち切られたキャラ |
| 担当編集フェイズ | 編集者と打ち合わせ |
| 持ち込みフェイズ | 編集部への持ち込み |
| 就寝フェイズ | 締切後の仮眠 |
| インスピレーション | 着想・ひらめき |
| 自信作 | 渾身の一作(リスク&リターン高) |
| アイディアフェイズ | ネタ出し |

### まだ世界観に乗っていない用語(変更検討)

| 現状 | 候補 | コメント |
|---|---|---|
| HP | **人気度** / 描き込み度 | L1-6 で逆 HP スケーリング検討中 |
| パワー(攻撃力) | **魅力** / 個性 | |
| バトル / 技 | **見せ場** / 名シーン | |
| 倒す / 没にする | **打ち切る** | |
| IP | **ひらめき** / 構想力 | |
| デッキ | **ネタ帳** | |
| 場 | **連載中の作品** / 執筆中 | |
| 進化 | **描き込む / 仕上げる** | |

## 1.3. 個別アイディア集(任意採用、参考)

提案 L1〜L3 に直接含めなかった、参考アイディア集:

### 「画材」をリソース化
- IP の代わりに「画材」(インク・トーン・ペン)を消費
- 種類ごとに使える効果が違う

### 「単行本化」ボーナス
- 原稿が 3 枚貯まったタイミングで「単行本ボーナス」発生
- 中間目標としてゲームに緩急

### 「読者人気投票」ミニフェイズ
- 毎ターン終了時、場のキャラ 1 体に「人気票」を獲得(+原稿 0.5)

これらは L2-2(編集者)や L2-4(打ち切り会議)で部分的に吸収済み。

# 第2部. 設計哲学 — 良い手塚カードゲームの条件

## 2.1. 良いカードゲームの 7 条件

| 条件 | 説明 | 反例 |
|---|---|---|
| **明快なコア・ループ** | 30 秒で「何をすればいいか」が分かる | 旧手塚: 6 フェイズ + 3 種類の効果カード = 初心者迷子 |
| **毎ターン悩ましい二択** | 戦略の核心。同じ判断を繰り返さない | 一本道のRPG的展開は飽きる |
| **試合時間 15-25 分** | 1 試合の体感が「短すぎない / 長すぎない」 | 遊戯王の 40 分は重い |
| **コア・ループ+独自要素** | 標準ルールに 1-2 個の独自性で個性を出す | 完全オリジナルは習得不能 |
| **逆転可能性** | 終盤まで諦めない設計 | 一度差がつくと挽回不能なゲームは退屈 |
| **デッキ構築の楽しさ** | 試合外の創造的楽しみ | 構築強制が強すぎると自由度ない |
| **テーマと整合性** | メカニクスがテーマを補強する | 「漫画家なのに HP 削り合い」が変 |

## 2.2. 現状の手塚カードゲームの強み・弱み再評価

### 強み(維持すべき要素)
1. **進化スタック保持** — Digimon と同等の堅実な機構、既に動いている
2. **原稿勝利条件** — 漫画家らしさの核心、絶対維持
3. **自信作リスク&リターン** — 独自性が光る数少ない要素
4. **泣きの一回** — 逆境からの復活、ドラマ性
5. **6 フェイズ(⓪〜⑤)** の枠組み — 漫画家の制作プロセスと対応

### 弱み(改善対象)
1. **コア・ループが不明瞭** — 「HP 削って原稿稼ぐ」が漫画家らしくない
2. **戦略の二択が無い** — 毎ターン同じ「キャラ出す→攻撃」の繰り返し
3. **試合時間 35-50 分** — 長すぎ
4. **戦闘が手動 HP 管理** — 計算ミスでテンポ悪化
5. **戦場が単一** — 場が 1 つだと選択肢が少ない
6. **ひらめき(IP)加速が遅い** — 序盤の停滞感
7. **効果カード(D)未整備** — 16 不明点、実装ボトルネック

## 2.3. 改善のフレームワーク

提案を 3 つの層に分類:

| 層 | 規模 | 内容 |
|---|---|---|
| **コア層(L1)** | 大改造、4 提案 | コア・ループそのものを再設計 |
| **手塚層(L2)** | 中規模、4 提案 | 漫画家でなければ表現できない独自要素 |
| **磨き層(L3)** | 小改造、4 提案 | 既存ルールへの小さな追加で改善 |

# 第3部. コア層(L1) — 根本的な再設計案

## L1-1. クエスト vs チャレンジ二択(Lorcana 派生)⭐⭐⭐

### コアコンセプト

> 「1 キャラを毎ターン、**執筆(原稿生産)** か **持ち込み(他キャラを倒す)** のどちらに使うか選ぶ」

これは Lorcana の Quest vs Challenge そのものを漫画家テーマに移植したもの。
**漫画家のリアル**: 連載原稿を書く時間を取るか、新連載の売り込みに行くか。両立は不可能。

### 具体ルール

**クエスト(=執筆)アクション:**
- 自陣の **レディ中**(縦向き)キャラを選択
- ひらめき(IP)を 1 消費
- そのキャラの **ロア値**(0.0-2.0、固有数値)を **自分の原稿に追加**
- そのキャラは **エグザート**(横倒し、次ターン使用不可)

**チャレンジ(=持ち込み)アクション:**
- 自陣の **レディ中** キャラを選択
- ひらめき(IP)を 2 消費
- 相手の **エグザート中** キャラを 1 体指定(レディキャラには挑めない)
- 互いに「魅力」分のダメージを「意思力」に与え合う(両者ダメージ、Lorcana 方式)
- 意思力 0 で打ち切り(没ゾーン送り)
- 自分のキャラが相手キャラを打ち切ったら原稿 +1(自信作なら +2)

### 戦略的駆け引きの構造

「クエストすると点取れるけど、エグザートして攻撃を受けるリスク」 vs 「チャレンジすると点取れないけど相手キャラ削れる」の二択。

これにより:
- 守備重視: クエストを我慢、レディ状態を保って相手のチャレンジを誘発
- 攻撃重視: 自分のキャラはチャレンジ専門に、相手のクエスト機を狩る
- 高速ロア取り: ロア値の高いキャラでクエスト連発、リスク承知

### なぜ手塚に最適か

| 漫画家の現実 | クエスト vs チャレンジでの表現 |
|---|---|
| 締切間際は執筆優先 | クエストでエグザート(動けなくなる) |
| 売り込みは作品を中断するリスク | チャレンジは点取れない |
| 連載中の作家は狙われる(他誌の引き抜き) | エグザート中だけチャレンジされる |
| 1 ターンに 1 キャラの全力投入 | 同じキャラを 1 ターンに使うのは 1 回 |

### 実装影響

- 「ロア値」フィールドを各キャラに追加(0.0-2.0)
- 「エグザート/レディ」状態の管理
- 「塗りたて(召喚酔い)」処理(出したターンは行動不可)
- バトルロジックを両者同時ダメージ式に
- UI: キャラを横倒し表示、ロア表示

実装規模: **大(20-25 時間)**、ただし手塚らしさへの貢献度は最大

### トレードオフ

✅ 戦略の核心が明快、漫画家らしさ最高、リプレイ性向上
❌ HP 概念がほぼ廃止、既存カードバランス全面再設計、エグザート UI 必要

## L1-2. 3 連載枠による並行戦線(Marvel Snap 派生)❌ 不採用(2026-05-13)

> **不採用理由**: ユーザー判断「複雑になりすぎる」。UI 改修コストとプレイヤー学習コストが大きく、現状の単一の場の方がシンプルに保てる。
> 雑誌方針による多様性は別の手段(例: 試合開始時に「特集テーマ」をランダム公開、全キャラに +N 効果)で代替検討。

旧仕様(参考):

### コアコンセプト

> 「漫画家は同時に複数の連載を抱える。場を 3 つの連載枠に分けて、それぞれ独立した魅力勝負」

Marvel Snap の 3 lanes をマンガ世界に翻訳。
**漫画家のリアル**: 看板作家ほど複数連載を抱える(手塚先生は最大 14 本同時連載)。

### 具体ルール

**場の構造:**
```
┌──────────┬──────────┬──────────┐
│ 連載枠 1 │ 連載枠 2 │ 連載枠 3 │
│ (雑誌X)  │ (雑誌Y)  │ (雑誌Z)  │
├──────────┼──────────┼──────────┤
│ 相手陣   │ 相手陣   │ 相手陣   │
│ ◇◇◇   │ ◇◇◇   │ ◇◇◇   │
├──────────┼──────────┼──────────┤
│ 自陣     │ 自陣     │ 自陣     │
│ ◇◇◇   │ ◇◇◇   │ ◇◇◇   │
└──────────┴──────────┴──────────┘
   各枠 3 体まで、計 9 体
```

**雑誌方針カード(ロケーション効果):**
試合開始時、3 枚の雑誌方針カードをランダム公開:
- 「医療誌」: 自信作キャラ +5 魅力
- 「少年誌」: 主人公キャラ +10 魅力
- 「青年誌」: 各ターン最高魅力キャラに「+1 ロア/ターン」
- 「学園誌」: Lv1 キャラ +3 魅力
- 「ホラー誌」: 没ゾーンが多い側のキャラ +1 魅力/没枚数
- 「歴史誌」: ターン終了時に場が空なら自動でキャラ登場
- 「ロマンス誌」: 同枠に異性ペア +5 魅力
- 「グルメ誌」: 場のキャラの魅力合計が低い側の +3 魅力(救済)

**勝敗:** 8 ターン以内に原稿 6 枚を先取、または 8 ターン終了時に原稿数で比較。

### 戦略的駆け引きの構造

- **集中投資**: 1 枠に全力、他 2 枠は捨てて 1 枠勝負
- **分散展開**: 3 枠均等に薄く広く、何枠か取れる
- **読み替え**: 雑誌方針を見て自分のデッキの強みが活きる枠に集中
- **諦め判断**: 不利な枠を意図的に捨てて他 2 枠に集中

### なぜ手塚に最適か

| 漫画家の現実 | 3 連載枠での表現 |
|---|---|
| 複数連載で「主戦場」と「副連載」がある | 3 枠で集中度が分かれる |
| 雑誌の方針(少年誌/青年誌)で評価が違う | 雑誌方針カードがそのまま機能 |
| 看板作家=複数誌で同時連載 | 3 枠全部に配置 |
| 「あの作品は打ち切り、こっちに集中」 | 1 枠捨てる戦略 |

### 実装影響

- 場を 3 レーン化(`state.playSession.sides[].lanes[3][3]`)
- 雑誌方針カードのプール(10-15 枚程度)とランダム公開システム
- UI: 3 列の場表示、雑誌方針表示
- カード配置先指定 UI

実装規模: **大(15-20 時間)**

### トレードオフ

✅ 戦略の幅が一気に広がる、リプレイ性最大、漫画家らしさ補強
❌ 場の UI 大改修、雑誌方針カードの新規作成、バランス調整必要

## L1-3. 8 ターン上限 + マナカーブ式リソース(Marvel Snap × Hearthstone 派生)⭐⭐

### コアコンセプト

> 「試合は 8 ターンで決着。毎ターンひらめきが +2 ずつ自動増加(=連載期間が長いほど作家の創作力が上がる)」

Marvel Snap の 6 ターン固定 + Hearthstone の自動マナ加算を融合。
**漫画家のリアル**: 連載が進むほどキャリアが積み上がり、大きな企画が打てるようになる。

### 具体ルール

**ターン上限:**
- 最大 8 ターンで強制終了
- ターン 8 終了時、原稿が多い方が勝者(=最終回サドンデス)
- ターン 6 終了時点で原稿 6 枚到達したら即勝利

**ひらめき(IP)成長:**
- ターン 1: 各プレイヤー ひらめき 2 でスタート
- 各ターン開始時に **自動 +2**(ONE PIECE DON!! 風)
- 上限なし、累積
- ターン 8 で最大 16 ひらめき

**ひらめきの用途:**
- キャラ配置(コスト 1-5)
- 進化(コスト 2-3)
- クエスト(コスト 1)
- チャレンジ(コスト 2)
- 効果カード使用(コスト変動)
- 主人公スキル発動(コスト 5-9)

### なぜ手塚に最適か

| 漫画家の現実 | このルールでの表現 |
|---|---|
| 新人時代は小ネタしか描けない | 序盤はコスト 1-2 のキャラのみ |
| ベテランになると大連載企画が通る | 終盤に高コスト・高魅力キャラ |
| 連載が終わるとき=最終回の盛り上がり | 8 ターン目=最終回フェイズ |
| 「次回作」の機会は限られる | デッキ切れリスク |

### 試合時間検算

| ターン | 1 ターン時間 | 累積時間 |
|---|---|---|
| 1-3 (序盤) | 60 秒 | 3 分 |
| 4-6 (中盤) | 100 秒 | 8 分 |
| 7-8 (終盤) | 150 秒 | 13 分 |

**合計**: 約 13 分 / 試合 → 目標達成

### 実装影響

- フェーズ進行に「ターン 8 で強制終了」ロジック追加
- ひらめきの自動加算処理
- ターン数 UI(現在何ターン目か)
- ターン 8 での勝者判定処理

実装規模: **中(8-12 時間)**

### トレードオフ

✅ 試合時間が予測可能、テンポ最高、序盤→終盤の盛り上がりが明確
❌ ターン 8 で終わるので「ひたすら殴り合う長期戦」は不可、ロングゲーム派には不向き

## L1-5. 主人公カードの「レベル制」廃止 → 固有能力制(One Piece × Lorcana × HS 派生)⭐⭐⭐

> ⚡ **2026-05-13 ユーザー提案**: 主人公のレベル(Lv1原案 → Lv2ネーム → Lv3清書)が必要に感じない。
> 代わりに、各主人公にデフォルト効果(常在)+ ひらめきコストで起動する固有スキルを持たせて、個性を立てたい。

### コアコンセプト

> 「主人公はレベルアップしない。代わりに各主人公に唯一無二の固有能力を持たせる」

これは One Piece(リーダー固定)/ Lorcana(ヒーロー固定)/ Hearthstone(ヒーロー固定)/ FAB(ヒーロー固定)全てに採用されている **現代 TCG の標準設計**。
進化させる楽しさは キャラクター(脇役 → 名脇役 → 相棒)に残し、主人公は **デッキ構築の起点** として位置づける。

### 具体ルール

**主人公カードの仕様変更:**
- 1 デッキ = 1 主人公(現状と同じ)
- レベル概念を **完全廃止**(Lv1/2/3 のカード分けを撤廃)
- 各主人公に以下の能力を持たせる:
  - **デフォルト効果**(常在能力、無コスト): 場に居るだけで永続効果
  - **固有スキル**(起動能力、ひらめきコスト): 1 ターン 1 回起動可能

### 主人公カード例(5 人で個性を立てる)

#### 例 1: ブラック・ジャック(医療系・サポート型)
- **デフォルト**: 自陣に医療ジャンルキャラが居る間、ターン終了時に **手札 +1**
- **固有スキル(コスト 5)**: 自陣キャラ 1 体の意思力を全回復、原稿 +0.5
- **戦略**: 医療キャラ中心のデッキ、長期戦で安定供給

#### 例 2: 鉄腕アトム(SF系・加速型)
- **デフォルト**: 毎ターン開始時、ひらめき +1(通常+2 と合わせて +3)
- **固有スキル(コスト 4)**: 自陣の SF ジャンルキャラ全員 +5 魅力(ターン中)
- **戦略**: SF キャラを大量展開、ひらめきブーストで高速プレイ

#### 例 3: 火の鳥(歴史系・復活型)
- **デフォルト**: 自陣キャラが没になる時、50% の確率で没ゾーンから手札に戻る(復活)
- **固有スキル(コスト 8)**: 自陣の没ゾーンから 1 体を場に復活(意思力半減)
- **戦略**: 持久戦・リソース循環

#### 例 4: サファイア(リボンの騎士、学園系・進化型)
- **デフォルト**: 自陣に Lv1(脇役)キャラが居る間、進化(描き込み)コスト -1
- **固有スキル(コスト 3)**: 山からキャラ 1 体を場に出す(コスト 4 以下、塗りたて)
- **戦略**: 大量進化、テンポデッキ

#### 例 5: メーテル(銀河鉄道999、SF系・コントロール型)
- **デフォルト**: 自陣に「謎」効果(=相手の手札 1 枚を裏向きで指定して没にできる)
- **固有スキル(コスト 6)**: 相手の場のキャラ 1 体をフリーズ(次のターン行動不可)
- **戦略**: 妨害・操作デッキ

### 戦略の変化

- **デッキ構築**: どの主人公を選ぶかで、デッキの方向性がほぼ決まる
  - ブラック・ジャック選択 → 医療ジャンル中心 + 長期戦カード
  - アトム選択 → SF ジャンル + 高速展開
- **対戦**: 相手の主人公を見て即座に戦略を予測 → メタ読み
- **試合中**: 主人公スキルの **発動タイミング** が戦略の節目

### 失われるもの・残るもの

| 旧仕様 | 新仕様 | 評価 |
|---|---|---|
| Lv1原案 → Lv2ネーム → Lv3清書 の段階進化 | 全て廃止 | 「描き込み」の楽しさは キャラクター進化に集中 |

- 主人公カードの **イラスト 3 種**(原案絵/ネーム絵/清書絵)としてフレーバーで残す
- ゲーム機構ではなく「コレクション要素」として
- 試合中はフレーバーテキストや絵柄の選択で楽しむ

### なぜ手塚に最適か

## L1-4. ピッチシステム(FAB 派生)✅ 実装済み(2026-05-13)

旧仕様(参考):

### コアコンセプト

> 「手札のカードは『プレイ用』にも『資源用(没にしてひらめき獲得)』にも使える二重価値」

FAB のピッチを手塚に翻訳。
**漫画家のリアル**: ボツになったネタはリサイクルしてアイデアの種にする。

### 具体ルール

**ピッチ値:**
- 全カードに「ピッチ値」(0/1/2/3)を記載

- **ピッチ済カードは没ゾーンへ**(復活可能、現状の没ゾーンと統合)

**ピッチタイミング:**
- 自分のターン中、任意のタイミング
- カードプレイのコスト支払い時にも使える
- 1 ターンあたりピッチ枚数の制限なし

**例:**
- ブラック・ジャック(コスト 3、ピッチ 1)
- ピノコ(コスト 1、ピッチ 3)
- ヒョウタンツギ(コスト 2、ピッチ 2)

ピノコ x2 をピッチ → 6 ひらめき → ブラック・ジャックをプレイ可能。

### 戦略的駆け引きの構造

- **強カード偏重デッキ**: ピッチ低めばかり、リソース枯渇しがち
- **バランスデッキ**: ピッチ中(2-3)中心、安定供給
- **トリッキーデッキ**: 強カード少数 + ピッチ高カード多数、燃料として

### なぜ手塚に最適か

| 漫画家の現実 | ピッチシステムでの表現 |

| 連載落としたネタを次の作品に活かす | カードを没にして次の燃料に |
| アイデア帳から1個削って大作品を描く | ピッチ→大コストキャラ |

## L1-6. 「原稿化中」リスク機構 + 逆 HP スケーリング(完全オリジナル)⭐⭐⭐⭐

> ⚡ **2026-05-13 ユーザー発想**: 「敵を倒したら原稿ゲージが上がる」のは普通のカードゲーム感、独自性ない。
> 倒しても没ゾーン送りにしかせず、自分のキャラを Lv2/Lv3 にして **自分で意図的に原稿ゾーンへ送る** ようにしたい。
> カードを横向きにして 1 ターン経過すれば原稿ゾーン到達、その間に攻撃されたら没ゾーンへ。
> Lv 上がると HP 増えるのではなく **逆に減る**(完成度高いほど脆い)などのリスク設計。

### コアコンセプト

> 「**漫画原稿は完成に近づくほど脆い**。守り抜いて掲載するか、踏み込まれて没るか」

これは **完全に手塚オリジナル** の発想。
他のどの TCG も「敵を倒す=点取り」モデルだが、これは「**自陣のキャラを完成原稿に変換**」モデル。
殴り合いではなく、**守って育てて発表する** ゲームへの転換。

### 漫画家の現実との一致

| 漫画家の現実 | 原稿化中リスク機構での表現 |
|---|---|
| ラフ(原案)は丈夫 — 何度でも描き直せる | Lv1 = HP 高(30) |
| ネームは細部まで決まる — 修正は痛い | Lv2 = HP 中(20) |
| 清書は完成寸前 — 一筆ミスで台無し | Lv3 = HP 低(10) |
| 完成原稿は印刷所に渡るまで脆い | 原稿化中の 1 ターン無防備 |
| 締切間際の事故で没原稿になる悲劇 | 原稿化中に攻撃されると没ゾーン |
| 「ボツになっても勉強になる(別ネタで使う)」 | 没ゾーンからリサイクル可(別効果カードで) |
| 倒した相手は単に「ボツ作品」 | 倒しても原稿にはならない(没ゾーンへ) |

### 具体ルール

#### キャラのレベルと HP の関係(逆スケーリング)

| Lv | 状態 | HP(意思力) | 魅力 | コスト | 原稿価値 |
|---|---|---|---|---|---|
| Lv1 | 脇役(原案) | **30**(高) | 5(低) | 1 | 0(原稿化不可) |
| Lv2 | 名脇役(ネーム) | **20**(中) | 10(中) | 2 | **+1 原稿** |
| Lv3 | 相棒(清書) | **10**(低) | 20(高) | 4 | **+2 原稿** |
| 自信作 | (各Lv 共通バフ) | HP +5 | 魅力 +5 | コスト +1 | 原稿価値 +1 |

**意図**: 強力なキャラほど脆い = 「glass cannon」設計。完成度が高いほどリスク。

#### 「原稿化中」アクション

**前提**: Lv2 または Lv3 のキャラのみ「原稿化」可能(Lv1 は不可、ラフは完成原稿になれない)

**手順**:
1. 自分のターン中、Lv2/Lv3 の自陣キャラを 1 体選択
2. ひらめき 2 を消費 (清書化作業のコスト)
3. **カードを横向きにする(エグザート)= 「原稿化中」状態**
4. 次の自分のターン開始時、そのキャラはまだ場にいれば → **原稿ゾーン送り、原稿カウンタ +1(Lv2) or +2(Lv3)**
5. ただし、原稿化中の状態でターン中に **意思力 0** にされた場合 → **没ゾーン送り、原稿は得られない**

#### 原稿化中キャラの扱い

- **無防備マーカー**: 場のキャラに「📝原稿化中」マーカー表示
- 相手のチャレンジ対象になり**やすい**(優先指定可)
- ただし「かばう」持ちが居れば守れる
- 原稿化中のキャラは攻撃側に回れない(エグザートしてるため)
- 効果カード(テコ入れ・アシスタント)で守ることも可能

#### ターン経過と完成

```
自分のターン T:
  - Lv3 キャラを「原稿化」(ひらめき2消費、エグザート)
  - 場に「📝原稿化中」マーカー付きで残る

相手のターン T+0.5:
  - 相手はマーカー付きキャラを集中攻撃可能
  - 攻撃される → 意思力削られる → 0 で没ゾーン
  - 生き残ったら → 次の自分のターン開始時に「完成」

自分のターン T+1:
  - 開始時、原稿化中キャラが場にいれば → 原稿ゾーンへ移動、+2 原稿
  - そのキャラはゲームから去る(原稿として保管)
```

### 戦略の変化(全面的)

#### 攻撃と守備の役割逆転

| 旧仕様 | 新仕様 |
|---|---|
| 攻撃 = 原稿稼ぐ | 攻撃 = **相手の原稿化を妨害**(直接得点にならない) |
| 防御 = 守るだけ | **守備 = 自陣の原稿化を完遂**(主たる得点源) |
| 場のキャラ多い = 攻撃力高い | 場のキャラ多い = **原稿化候補の数** |
| 高 Lv = 強い | 高 Lv = **脆い、原稿価値高い** |

#### 新たな駆け引き

1. **タイミング読み**: 相手の原稿化を予期して攻撃カードを温存
2. **囮戦略**: Lv2 を多数原稿化して相手の攻撃を分散 → 本命の Lv3 を守る
3. **特攻戦略**: 自信作 Lv3 を一気に原稿化 → 相手のリソースを誘発、その隙に別キャラを成長
4. **遅延戦略**: 原稿化を控えて守備固め、後半に一気に投入
5. **ラッシュ戦略**: 序盤の Lv1 を急速に Lv2 化 → 数で押す

### 倒したご褒美(代替案)

「倒しても原稿は得られない」だけだと、攻撃する動機が薄い。代替報酬を設定:

#### 代替案 A: 相手の没ゾーン操作権
- 倒した側は、相手の没ゾーンから 1 枚を **没ゾーンの底に送る**(復活妨害)
- リサイクル不可能なボツに

#### 代替案 B: ひらめき強奪
- 倒した側は、ひらめき +1 獲得(=「相手の作品で得たインスピレーション」)

#### 代替案 C: 原稿化妨害ボーナス
- **原稿化中のキャラを倒した場合のみ** 攻撃側にボーナス: ひらめき +2 + 1 ドロー
- 通常のキャラ討伐は報酬なし

#### 代替案 D: 手札へ「他誌のヒット作」を盗作
- 倒した相手のキャラを **自分の手札に加える**(コスト0で再使用可、漫画家の業界感)
- 「他誌のヒット作からインスパイア」演出
- これは大胆だが面白い

**推奨**: 代替案 C(原稿化中のキャラを倒した場合のみボーナス)が一番健全。
攻撃が「原稿化阻止」に意味を持つ、純粋な攻撃は割に合わない設計。

### 関連派生案(さらに大胆なアイデア)

#### 派生 1: 「校了」(けいりょう)ターボアクション
- 通常: Lv2/Lv3 を原稿化 → 1 ターン待機 → 原稿

## L1-7. HP 問題への抜本解決案(2026-05-13 テストプレイ受け、HP 数値管理問題への複数アイディア)⭐⭐⭐⭐

> ⚡ **2026-05-13 テストプレイで判明**: 主人公固有能力化 + 覚醒メカ + ピッチを実装した新ルールでも、

> 計算ミス、紙片メモ、思考フロー停止の元になっている。

### HP 問題の本質

| 問題 | 影響 |
|---|---|
| 手動計算 30-90 回 | テンポ著しく阻害、試合時間延長 |
| HP 数値が幅広い (10-70) | 覚えにくい、誤計算 |
| ダメージ量が不明瞭 | 攻撃力 = 魅力? それとも別? カード依存? |
| キャラごとに HP が違う | 全体把握困難 |
| 紙片メモが必要 | デジタル UI と相性悪い |

### 10 のアイディア(順不同、ボールド発想含む)

#### A. 描き込み度ゲージ(完成度バー)⭐⭐⭐⭐ 推奨第 1 候補

**コンセプト**: HP 廃止、代わりに「完成度」(0-100%)が増えていく。

- 場に出した瞬間: 完成度 0%(下書き)
- 自分のターン終了時に **自動 +20%**(描き進める)
- ひらめき消費で追加 +30%(集中執筆 / 描き込み)
- **100% 到達 → 自動的に原稿ゾーンへ送られる(原稿 +1)**
- 攻撃を受けると **-50%**(描き直し、ネーム破棄)
- Lv が上がる(進化)と +20% ボーナス
- 0% に戻されたキャラは没にはならず、また描き直し

**なぜすごいか**:
1. HP の数値管理が消える(0-100% の進捗バーのみ)
2. 「原稿の獲得手段」が漫画家らしく再設計される
3. ターン経過で着実に進捗 → 「育てる楽しさ」
4. 攻撃の意味: 「相手の原稿完成を妨害」 = 漫画家業界で言う「ライバル誌に書き手を取られる」感
5. 「倒すと原稿」の陳腐感を完全脱却

**漫画家のリアル一致**:
- 連載中の作品: 完成度がじわじわ上がる
- 締切間近: 集中執筆で一気に完成
- アクシデント(編集没出し / 描き直し): 進捗後退

**実装影響**: 中規模(15-25 時間)
- HP/ダメージロジック削除
- `inst.progress` (0-100) フィールド追加
- ターン終了時の自動 +20% 処理
- 100% で原稿ゾーン自動移動 + 原稿カウンタ +1
- UI: 進捗バー(視覚的に分かりやすい)

#### B. 紙(原稿用紙)の枚数式 ⭐⭐⭐ シンプル最強候補

**コンセプト**: 各キャラに「紙」(1-3 枚)。攻撃で 1 枚破れる、0 枚で没。

- 場に出した時、キャラ固有の「紙の枚数」(1-3、自信作は 3)を載せる
- 攻撃を受けるたびに 1 枚破れる(物理的にトークンを 1 個取る)
- 0 枚で没ゾーン
- 計算ゼロ、トークン or 数字カウンタのみ

**メリット**:
- 究極のシンプル、計算ゼロ
- 「漫画原稿」テーマと完全一致
- 自信作の差別化が直感的(紙 3 枚 vs 1 枚)
- UI 実装が最も軽い

**デメリット**:
- 攻撃ダメージの差異(強キャラの攻撃 vs 弱キャラの攻撃)が消える
- 細かい駆け引きが減る

**実装影響**: 小(5-10 時間)
- HP/ダメージ廃止、`inst.papers` (1-3) で管理
- 攻撃成立で -1、0 で没ゾーン

#### C. ヒット式(回数制)⭐⭐ B と類似

各キャラに「ヒット耐久値」(1-3 程度)、攻撃 1 回 = 1 ヒット消費、規定回数で没。
B と本質同じだが、「ヒット」表記の方がカードゲーム慣れしてる人に通じる。

#### D. ジャンクション値(ピンチ値、MTG ダメージカウンタ風)⭐⭐

0 から開始、攻撃で +1 蓄積、5 で没。
カードに「耐久 5」と書いてカウンタを載せる方式。
B / C の逆方向版。

#### E. 連載人気度ゲージ(増減型、Hearthstone like)⭐⭐⭐

**コンセプト**: 「人気度」が増減する。0 で打ち切り、20 で看板昇格。

- 各キャラに「人気度」初期 10
- 攻撃で -X(攻撃側の魅力分)
- 効果カード「広告」「アシスタント」で +X
- ターン経過で +1 自動(連載継続ボーナス)
- 0 で打ち切り(没)
- **20 到達で「看板昇格」**: 原稿 +1 ボーナス、以降人気 10 にリセット

**メリット**:
- 双方向の増減で駆け引き深い
- 「看板昇格」が明確な目標
- 漫画家のリアル(人気が上下する)

**デメリット**:
- 結局数値管理が必要(HP 問題が部分的にしか解決しない)

#### F. HP 廃止 → 魅力比較バトル ⭐⭐⭐⭐ 第 2 候補

**コンセプト**: 戦闘は魅力 vs 魅力、高い方の勝ち、負けた方は没。

- バトル時: 攻撃側魅力 vs 防御側魅力
- 高い方が勝ち、負けた方は **即没ゾーン**
- 同値は両者没(相打ち)
- ひらめき消費で魅力 +5 ブースト可
- Lorcana / Vanguard 風

**メリット**:
- 計算 1 回、超シンプル
- テンポ最高
- 「魅力対決」が漫画キャラ感

**デメリット**:
- 一撃必殺で長期戦不可
- 「タフな脇役」概念が消える

#### G. ダイス判定(運要素追加)⭐⭐

魅力 + d6 ダイスロールで対決。ジャイアントキリングが起きる。
開運コロシアム風。但し運要素好き嫌いが分かれる。

#### H. 連載期間ベース耐久(L2-1 連動)⭐⭐

連載期間で耐久が動的決定:
- 1-2 ターン経過: 紙 1 枚
- 3-4 ターン: 紙 2 枚
- 5+ ターン: 紙 3 枚

長く守るほど強くなる、連載期間カウンタ(L2-1)を活用。

#### I. 没までの「描き残し原稿」式(One Piece ライフ風)⭐⭐⭐

**コンセプト**: 各キャラに「描き残し原稿」(2-5 枚)、攻撃でデッキトップ 1 枚が没。

- キャラに「描き残し」(2-5)を載せる
- 攻撃を受けると、自分のデッキトップ 1 枚を **没ゾーンへ送る**(描き残し -1)
- 描き残し 0 でそのキャラ自体も没
- 没ゾーンに送られたカードの IP が即獲得(ピッチと連動)

**メリット**:
- 攻撃が「相手のデッキを削る」二重効果
- ピッチシステムと相性抜群
- 「描いてもボツになる」漫画家あるある

**デメリット**:
- やや複雑、デッキ切れリスク

#### J. 「制作中 / 完成」二状態スイッチ式 ⭐⭐

キャラは「制作中」(攻撃で即没) or 「完成」(1 攻撃耐える) の二状態のみ。
進化で完成へ移行。HP の代わりに 2 値スイッチ。
シンプルだが単調。

---

### 推奨ベスト解: A + F のハイブリッド ⭐⭐⭐⭐⭐

**「描き込み度ゲージ + 魅力比較バトル」の融合**:

1. **HP 廃止、すべて「完成度」(0-100%) で管理**
2. **バトル時 = 魅力比較**:
   - 攻撃側魅力 vs 防御側魅力
   - 攻撃側勝ち: 防御側の **完成度 -50%**(0% 未満は 0% に)
   - 防御側勝ち: 攻撃側のキャラを **没ゾーン送り**(攻撃成功してない場合のペナルティ)
   - 同値: 両者 -30% (相打ち)
3. **完成度の上昇**:
   - 自分のターン終了時に自動 **+20%**
   - ひらめき消費で **+30%**(集中)
   - 進化(描き込み)で **+30%**(描き加える)
4. **完成度 100% 到達** → 即時 **原稿ゾーンへ移動、原稿 +1**(看板キャラなら +2)
5. **没条件の見直し**:
   - 完成度 0% でも没にはならない(描き直し)
   - 没ゾーン送りは **効果カード**(=ボツ出し) or **デッキ切れ**(=描き残し 0) or **相打ち**(=魅力比較負け)
   - これにより「攻撃で倒して原稿稼ぐ」陳腐性を完全脱却

**この設計の偉さ**:
- HP 計算ゼロ(進捗バーのみ視覚化)
- 漫画家らしさ最大化(描き進めて完成、攻撃 = 妨害)
- 「攻撃の意味」が再定義される(直接得点でなく相手の妨害)
- 完成度ゲージで「育てる楽しさ」発生
- 試合時間予測: 40 分 → **20-25 分**(計算時間消滅)

**実装影響**: 中規模(20-30 時間)
- HP/ダメージ管理ロジック削除
- `inst.progress` (0-100) フィールド追加 + UI 進捗バー
- バトルロジック書き換え(魅力比較のみ)
- 100% 到達で原稿ゾーン自動移動 + 原稿カウンタ +1
- ターン終了時 +20% 自動処理

---

# 第4部. 手塚層(L2) — 漫画家でなければ表現できない独自メカニクス

## L2-1. 連載期間カウンタ + 看板昇格 ⭐⭐⭐

### コアコンセプト

> 「キャラは場に長く居続けると強くなる。でも一定期間で人気が頭打ち→マンネリ化」

これは現実の人気漫画の現象を直接ルール化したもの。
他のどの TCG にも無い、**手塚オリジナルメカニクス**。

### 具体ルール

各キャラインスタンスに `serial_turns`(連載ターン数)を持たせる:

| 連載ターン | 状態 | 効果 |
|---|---|---|
| 0 (出したターン) | 新連載 | ロア値 0(クエスト不可) |
| 1-2 ターン | 連載中 | ロア値 1.0 |
| 3-4 ターン | 人気作 | ロア値 1.5、毎ターン自動 +0.5 原稿 |
| 5-6 ターン | 看板 | ロア値 2.0、毎ターン自動 +1.0 原稿 |
| 7+ ターン | マンネリ | ロア値 0.5、攻撃時 -5 魅力 |
| 8+ ターン | 打ち切り危機 | ターン終了時に自動没 |

進化(描き込み)するとカウンタリセット = 新装版扱いで再スタート。

### 戦略的駆け引きの構造

- **看板狙い**: キャラを 5-6 ターン守り通して看板化、原稿生産マシンに
- **打ち切り回避**: マンネリ前に進化してリセット
- **短期決戦**: マンネリ前に高速で原稿稼ぐ
- **敵看板を打ち切る**: 相手の看板キャラを優先チャレンジで没

### なぜ手塚に最適か

| 漫画家の現実 | 連載期間カウンタでの表現 |
|---|---|
| 新連載は地味 | 0 ターン目はロア 0 |
| 人気作になると単行本売上爆発 | 看板=自動原稿生産 |
| 長期連載でマンネリ化 | 7+ターンで弱体化 |
| リブート・新装版で人気再燃 | 進化でカウンタリセット |
| 「鉄腕アトム」「ブラック・ジャック」のような看板作 | 看板状態のキャラ |

### 実装影響

- インスタンスに `serial_turns` フィールド追加
- ターン終了時にカウンタ更新、状態判定
- UI: キャラの上に連載ターン数表示、状態アイコン(🆕新連載/⭐人気作/👑看板/😩マンネリ)
- 状態によるロア値・魅力・効果の動的計算

実装規模: **中(8-12 時間)**

### トレードオフ

✅ 完全オリジナル、漫画家らしさ最高、「育てる楽しさ」追加
❌ カウンタ管理が UI で煩雑、初心者の学習コスト
❌ L1-1(クエスト) と組み合わせる必要(ロア値の動的変化)

## L2-2. 編集者(担当編集)カードシステム ⭐⭐

### コアコンセプト

> 「担当編集者がキャラの行動を指示・修正できる。場に編集者を出すと自分のキャラがブーストされる」

現状の「担当編集フェイズ」(④フェーズ)を実体化させる新カードタイプ。
**漫画家のリアル**: 担当編集の手腕で漫画の出来は大きく変わる。

### 具体ルール

**「編集者カード」(新カードタイプ):**
- 場に 1 名まで配置可能(連載枠とは別枠)
- コスト: 2-4 ひらめき
- 各編集者に固有スキル(1 ターン 1 回起動可)

**編集者カード例:**

| カード | コスト | スキル |
|---|---|---|
| ベテラン編集 | 3 | 自陣キャラ 1 体の魅力 +5(ターン中) |
| 厳しい編集 | 4 | 相手キャラ 1 体の連載期間 -1(マンネリ加速) |
| 新人編集 | 2 | 自分の手札 +1 ドロー |
| 名物編集 | 5 | このターン、自分のクエスト報酬 +0.5 ロア |
| 海外編集 | 3 | 雑誌方針カードを1枚交換できる |

**編集者の交代:**
- 各ターン 1 回、手札の編集者カードと場の編集者を交代可能
- 古い編集者は没ゾーンへ

### 戦略的駆け引きの構造

- **ターン中の戦力修正**: 編集者スキルで魅力ブーストして相手のチャレンジを撃退
- **相手のキャラ弱体化**: 「厳しい編集」で相手の看板を打ち切り促進
- **デッキ加速**: 「新人編集」でドロー、長期戦に強い

### なぜ手塚に最適か

| 漫画家の現実 | 編集者システムでの表現 |
|---|---|
| 担当編集の影響力(松本零士「銀河鉄道999」の故 内田勝編集など) | 編集者ごとに違うスキル |
| 編集者交代で作風変化 | カード交代で戦略変化 |
| 「あの編集はうるさい」 | 厳しい編集が相手キャラを弱める |
| 「コミックス担当が優秀」 | 名物編集でロア +0.5 |

### 実装影響

- 新カードタイプ `editor` の追加
- 場に編集者用ゾーンを 1 枠追加
- 編集者スキルの起動 UI
- カードプール(編集者カード 5-10 枚作成)

実装規模: **中(10-15 時間)**

### トレードオフ

✅ 漫画家世界観の補強、戦力修正の戦略性、現「担当編集フェイズ」を実体化
❌ 新カードタイプの追加で UI 複雑化、編集者カードを誰がどう作るか(キャラ作成タブで対応)

## L2-3. ジャンル相性システム ⭐

### コアコンセプト

> 「キャラと雑誌のジャンルが一致するとブースト。手塚作品の多ジャンル性を反映」

L1-2(3 連載枠) と連動。雑誌(ロケーション)とキャラのジャンルマッチングで効果。

### 具体ルール

**ジャンル(全 8 種類):**
- 医療 / SF / 歴史 / 学園 / ホラー / 恋愛 / グルメ / ファンタジー

**カードのジャンル属性:**
- 各キャラ・効果カードに「ジャンル」を 1-2 個設定(複数ジャンルあり)
- 例: ブラック・ジャック = 医療
- 例: アトム = SF
- 例: 火の鳥 = 歴史/SF

**雑誌方針との連動:**
- 雑誌方針カードに「主軸ジャンル」を設定
- 例: 医療誌の主軸ジャンル = 医療
- ジャンルが一致するキャラは魅力 +3、ロア +0.5

**シナジー:**
- 同じ枠に同ジャンルのキャラが 2 体以上 → さらに +5 魅力(連載作品グループ)
- 異なる枠に同ジャンル → +1 ひらめき/ターン(専門特化作家)

### 戦略的駆け引きの構造

- **専門特化デッキ**: 1 ジャンル中心、シナジー最大化
- **オールラウンダー**: 多ジャンルキャラ(火の鳥など)を活用
- **メタ読み**: 相手のジャンル予想して妨害ジャンルを狙う

### なぜ手塚に最適か

| 漫画家の現実 | ジャンル相性での表現 |
|---|---|
| 手塚作品は医療(BJ)/SF(アトム)/歴史(火の鳥)等多岐 | ジャンル属性で表現 |
| 漫画家は得意ジャンルに特化することも多い | 専門特化デッキ |
| 「BJはやっぱり医療誌」 | ジャンル一致でブースト |

- 雑誌方針カードの主軸ジャンル設定

実装規模: **中(8-10 時間)**

❌ カード全体にジャンル属性を後付けする作業(50 枚以上)、シナジー判定が複雑化

## L2-4. 打ち切り会議イベント ⭐

### コアコンセプト

> 「定期的に編集会議が開かれ、人気のないキャラが打ち切られる。マンネリ化したキャラへの圧力」

現実の漫画雑誌の「アンケートで下位作品打ち切り」を再現。

### 具体ルール

**発動条件:**
- 試合の **ターン 4 と ターン 6** に自動発動
- 各プレイヤーの場の全キャラを対象に「人気投票」

**人気投票:**
- 場の全キャラの「魅力 + ロア値」を計算
- 各プレイヤーの最も低いキャラ 1 体が **自動的に没ゾーンへ**(打ち切り)
- 該当キャラが複数あれば、所有プレイヤーが選ぶ
- 主人公・自信作は対象外(保護)
- 「かばう」持ちは対象外

**追加効果:**
- 打ち切られたキャラのオーナーは原稿 -0.5(連載失敗ペナルティ、最低 0)
- 生き残った最高魅力キャラには「ヒット」マーカー = 次ターンの魅力 +5

### 戦略的駆け引きの構造

- **打ち切り回避**: 弱キャラを意図的に進化(描き込み)して魅力底上げ
- **打ち切り誘導**: 相手の弱キャラに集中チャレンジで打ち切り候補を増やす
- **タイミング読み**: ターン 4 / 6 直前にキャラ展開・進化のラッシュ

### なぜ手塚に最適か

| 漫画家の現実 | 打ち切り会議での表現 |
|---|---|
| 雑誌のアンケート最下位は打ち切り危機 | 最低魅力キャラが自動没 |
| 手塚先生も若手時代に打ち切り経験あり | 試合中の演出としてリアル |
| 「あの作品は人気がない」 | 数値で表現 |
| ベテランの看板作品は保護される | 主人公・自信作・かばう持ちは対象外 |

### 実装影響

- ターン 4/6 で自動発動するイベント処理
- 最低魅力キャラの自動判定・除外ロジック
- 「ヒット」マーカーの実装
- UI: 打ち切り会議の演出表示

実装規模: **小〜中(5-8 時間)**

### トレードオフ

✅ 漫画家らしさ強烈、定期的なボードリセット効果(膠着防止)、戦略の幅広がる
❌ 「自動で削られる」感が好きじゃないプレイヤーもいる、保護対象の調整必要

# 第5部. 磨き層(L3) — 既存ルールへの小改善

## L3-1. 進化時 1 ドロー + 戦吼効果(Digimon × Hearthstone)

### 具体ルール

進化(描き込み)成功時に **自動ボーナス**:
1. 山から 1 枚ドロー(Digimon 式)
2. キャラの **進化時効果**(=戦吼)発動

進化時効果の例:
- ブラック・ジャック Lv2(ネーム): 自陣キャラ 1 体に +5 魅力
- ブラック・ジャック Lv3(清書): 相手キャラ 1 体に -3 魅力 + 1 ドロー
- アトム Lv2: 自分の原稿 +0.5
- ピノコ Lv2: 相手の手札 1 枚をランダム捨て

### なぜ良いか

- 進化への動機付け強化(現状は IP 消費するだけで地味)
- カードごとの個性が出る
- Hearthstone Battlecry の楽しさを取り込む

### 実装規模: 小(3-5 時間)

---

## L3-2. 「描き込み」連鎖ボーナス

### 具体ルール

同ターン中に **連続して進化を実行** すると、2 段目以降にボーナス:
- 2 連続進化: +1 ドロー
- 3 連続進化: +1 ドロー + 1 ひらめき
- 4 連続進化: +1 ドロー + 1 ひらめき + 原稿 +0.5

### なぜ良いか

- 進化を 1 ターンに複数行う動機(現状は手間と感じることも)
- 「ノリに乗った漫画家が一気に描き上げる」演出
- リソースを温存して爆発的な進化ターンを作る戦略

### 実装規模: 小(2-3 時間)

---

## L3-3. 読者反応トリガー(Vanguard 派生)

### 具体ルール

原稿獲得時、**山トップを 1 枚めくる** → 描かれた「読者反応アイコン」で効果発動:

| アイコン | 効果 | 頻度 |
|---|---|---|
| 🔥 熱狂 | 自陣全員 +2 魅力(1 ターン) | 普通 |
| 💎 名作 | 原稿 +0.5(追加) | レア |
| ⚡ 賛否両論 | 相手の手札を 1 枚ランダム見る | 普通 |
| ⭐ 発見 | 1 ドロー | 普通 |
| アイコン無し | 効果なし | 約 40% |

めくったカードは没ゾーンへ。

### なぜ良いか

- 原稿獲得が単なる数値変動でなく「事件」に
- 山めくりのドラマ性、Vanguard の楽しさ
- 「読者が反応した!」演出が漫画家テーマと一致

### 実装規模: 中(5-8 時間)、全カードにアイコン追加が必要

---

## L3-4. アシスタント雇用効果カード

### 具体ルール

新しい効果カードタイプ「アシスタント」:
- コスト 2-4 ひらめき
- 効果は「ターン中の単発ブースト」
- 使用後は没ゾーンへ

アシスタントカード例:
- 「ベタ塗りアシ」: 自陣キャラ 1 体に +3 魅力(ターン中)
- 「背景アシ」: 連載枠を 1 つ選び、その枠のキャラ全員 +2 魅力
- 「ペン入れアシ」: 自陣キャラ 1 体を即進化(描き込み完了)
- 「ネタ出しアシ」: 2 ドロー
- 「徹夜アシ」: 1 ひらめき消費して原稿 +0.5

### なぜ良いか

- 効果カード D カテゴリ(展開アイディア)を具体化
- 「クライマックスの瞬間にアシ呼んで仕上げる」演出
- カードバリエーション増加

### 実装規模: 小(3-5 時間)、カードプール作成のみ

# 第6部. 避けるべき設計(アンチパターン)

旧 18 提案や TCG 分析から、**手塚カードゲームに合わない**設計を明示:

## アンチ 1. メモリーゲージ式 IP 共有

- 旧提案 16
- Digimon の独自リソース、確かに面白いが**学習コスト極めて高い**
- 「マイナス側で渡される」概念は直感的でない
- 手塚はもっとシンプルに

## アンチ 2. チェーン・スタック多段発動

- 旧提案 5
- 遊戯王のチェーン処理を模倣する案、ただし**初心者大混乱**
- 1 ターン解決時間が極端に長くなる
- 単純な「攻撃→反応」程度で十分

## アンチ 3. 主人公への直接攻撃で勝利

- 旧提案 6
- ライフ削り型の勝利条件追加だが、**主人公の世界観が「触れられない神格」とズレる**
- 原稿勝利 1 本に絞った方が漫画家らしい

## アンチ 4. 細かい HP の数値管理

- 現状の手動管理
- 計算ミスが起きやすく、UI 上の煩雑さもある
- 「魅力 vs 魅力」の比較に置き換える方がスマート

## アンチ 5. デッキの巨大化(60-80 枚)

- 現状のルール
- カードを引く運要素が大きくなり、戦略が無意味化
- 50 枚以下にして引きを安定させる

## アンチ 6. 1 ターンに複数攻撃の繰り返し

- 旧の「場の全員が1回ずつ攻撃」案
- 1 ターンが長くなり、判断疲れ
- L1-1(クエスト vs チャレンジ) で「1 キャラ 1 アクション」に絞る

## アンチ 7. 完全運ゲー要素

- 開運コロシアムのジャンケンを丸ごとコピー
- 確かに楽しいが、**戦略性を求める層には不評**
- L3-3(読者反応トリガー) 程度の運要素で十分
