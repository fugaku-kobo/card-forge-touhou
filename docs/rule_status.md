# 📋 第二稿の核心

# A. 勝敗条件

| 項目 | ルール |
|---|---|
| **勝利条件** | 原稿エリアに **6 枚** 溜まると作品完成(勝利) |
| **敗北条件** | 泣きの一回を使った状態で自陣のキャラがすべて没 |
| **デッキ切れ** | 没案ゾーンをシャッフルしてデッキへ再生(敗北はない) |
| **引き分け** | 存在しない |

# B. デッキアイコンカード(旧主人公カードの代替)

## B.1. 基本仕様

| 項目 | 内容 |
|---|---|
| **役割** | デッキの中身を固定する識別カード |
| **進化** | なし(Lv 概念廃止) |
| **配置** | 専用ゾーンに 1 枚固定、ゲーム中ずっと表向き |
| **デッキ枚数カウント** | デッキに含まれない別カード |

## B.2. 能力構造

各アイコンカードに 3 種の要素:

| 要素 | 説明 |
|---|---|
| **両面共通効果** | 場(アイコンゾーン)にいる間、常時発動するパッシブ能力 |
| **裏面効果(覚醒)** | 裏返し時に発動可能な強力な効果 |
| **裏面コスト** | 裏面効果を使う際の IP コスト |

## B.3. 裏返し(覚醒)ルール

- 発動条件: **対戦相手の原稿が 4 枚以上** に到達した時点で覚醒可能
- 自分のターン中、アイコンカードを物理的に裏返す
- 裏返し後、1 回だけ次のいずれかを使える:
  - **このカードの裏面効果**(コスト払って発動)
  - **どんでん返しカード**(別山から 1 枚選択、コスト払って発動)
- 一度裏返したら元に戻らない

## B.4. 例(Excel から)

| カード | 両面共通効果 | 裏面効果 | コスト |
|---|---|---|---|
| ブラック・ジャック | ドローフェイズに自陣キャラから 10 ダメージ除去 | 全ダメージ除去+相手全キャラ合計 200 ダメ | 10 |
| 鉄腕アトム | 自陣キャラの被ダメ -10 | アトム 1 没→相手全キャラ 150 ダメ | 10 |
| ナギ(火の鳥) | 自陣キャラの与ダメ +10 | 10 ドロー+進化不要で展開し放題 | 2 |

# C. キャラクターアイディアカード(クリーチャー)

## C.1. 3 階層

| レベル | 通称 | 説明 |
|---|---|---|
| Lv1 | **原案** | 場に直接出せる、進化前提 |
| Lv2 | **ネーム** | 原案の上に重ねて進化 |
| Lv3 | **清書** | ネームの上に重ねて進化 |

## C.2. ステータス

| 項目 | 内容 |
|---|---|
| **HP** | 受けた技ダメージ合計が HP を超えたら没 |
| **没で得る IP** | 没案ゾーン送りで獲得するインスピレーションパワー(1〜3) |
| **イチオシ** | フラグ(旧「自信作」)、HPが多めで強力、没で相手の原稿 +2 |
| **パッシブ能力** | 場にいる間に発動する固有効果(任意) |
| **技 1 / 技 2** | 持ち込みバトル時に IPコスト払って発動する攻撃技 |

## C.3. 進化(練り上げ)

- 同名キャラで Lv 順正しく重ねる
- 場に出て **1 ターン経過後** に進化可能
- 進化したターンに攻撃可
- 進化スタックは保持(下のカードは積まれる)

## C.4. 場の上限

- **3 枚まで**(旧 6 枚から大幅縮小)
- 上限到達時、新キャラ出せない

---

# D. 効果カード(3 種類)

## D.1. 展開アイディアカード

- 「魔法カード」「サポートカード」相当
- ② アイディアフェイズ または ④ 担当編集フェイズで使用
- フィールド: カード名 / 没で得るIP / コスト / カード効果

## D.2. テコ入れカード

- 「罠カード」相当
- ② または ④ で **伏せて配置**
- 相手のターンに発動可
- 主に持ち込みバトル時の反撃・防御
- フィールド: カード名 / 没で得るIP / コスト / カード効果

## D.3. どんでん返しカード

- アイコンカードの **裏返し(覚醒)時のみ** 使用可
- **別山(0〜5 枚)** に置く、メインデッキとは別
- コストは 9 で統一(暫定)
- フィールド: カード名 / 没で得るIP / コスト(=9) / カード効果

---

# E. 持ち込みバトル(③ 持ち込みフェイズ)

| 項目 | ルール |
|---|---|
| **発動** | 相手のキャラを指定して持ち込みバトルを仕掛ける |
| **コスト** | キャラの技コスト(IP)を払う |
| **ダメージ計算** | 受けた技の合計値 が HP を超えると没 |
| **没時の処理** | 没にされたキャラは **インスピレーションゾーン** へ(没案ではない) |
| **原稿獲得** | キャラ 1 枚没にするごとに、**デッキトップから 1 〜 2 枚** を原稿ゾーンへ表向きで送る |
| **イチオシ没** | デッキトップから **2 枚** が原稿に(通常は 1 枚) |
| **先攻 1 ターン目** | このフェイズスキップ |

---

# F. ひらめき(IP)/ インスピレーションパワー

## F.1. 取得経路

| 経路 | 説明 |
|---|---|
| **没案ゾーン送り** | インスピレーションゾーンのカードを没案ゾーンに送る → そのカードの「没で得る IP」を獲得 |
| **インスピ送り** | アイディアフェイズ / 担当編集フェイズ / 就寝フェイズで手札を任意でインスピレーションゾーンへ |
| **就寝クリンナップ** | 手札上限 5 枚調整で送られる |

## F.2. 使用

- 技の発動
- カードプレイのコスト
- 上限: なし(累積可)

---

# G. ゾーン構造(2 つの "墓地系")

| ゾーン | 役割 |
|---|---|
| **インスピレーションゾーン** | バトル没になったキャラ、就寝クリンナップの手札を一時置く場 |
| **没案ゾーン** | 本当の墓地、インスピレーションから送ると IP 獲得、デッキ切れでシャッフル元 |
| **原稿ゾーン** | キャラ没時にデッキトップが送られる勝利ポイント置き場 |
| **キャラクターアイディアゾーン** | 場(上限 3 枚) |
| **アイコンゾーン** | デッキアイコンカード固定 |
| **手札** | 上限 5 枚 |
| **デッキ** | メインデッキ(60〜80 枚) |
| **どんでん返しデッキ** | 別山(0〜5 枚)、覚醒時参照 |

---

# H. ゲーム進行(6 フェイズ)

| # | フェイズ | 内容 |
|---|---|---|
| ⓞ | **開始フェイズ** | 裏向きのキャラクターカード(セットアップで伏せた原案)を表に |
| ① | **ドローフェイズ** | デッキから **2 枚ドロー**(先攻 1 ターン目はスキップ) |
| ② | **アイディアフェイズ** | 原案カードを出したいだけ場へ(上限 3)、進化、展開カード使用、テコ入れ伏せ、相手原稿 ≥4 でどんでん返し使用可 |
| ③ | **持ち込みフェイズ** | 持ち込みバトル(先攻 1 ターン目はスキップ) |
| ④ | **担当編集フェイズ** | 展開カード使用、テコ入れ伏せ |
| ⑤ | **就寝フェイズ** | 手札上限 **5 枚** までインスピレーションゾーン送り or ドロー、ターン終了 |

# I. ゲーム開始(セットアップ)

| 手順 | 内容 |
|---|---|
| 1 | デッキアイコンカードを設置(アイコンゾーン) |
| 2 | デッキを互いにシャッフル後、初期手札 **7 枚** ドロー |
| 3 | マリガンチェック(下記) |
| 4 | 原案カード 1 枚をキャラクターアイディアゾーンに **伏せて** 設置 |

## マリガンルール

- 初手に **原案キャラクターアイディアカード** がない場合 → マリガン可
- 手札公開 → シャッフル → 7 枚引き直し
- 回数ペナルティあり:
  - 1 回目: ペナルティなし
  - 2 回目: 引き直した 7 枚のうち 1 枚をデッキボトムへ
  - 3 回目以降: 戻すカードを +1 枚ずつ(3 回目=2 枚、4 回目=3 枚...)

---

# J. 泣きの一回

| 項目 | 内容 |
|---|---|
| 発動条件 | 持ち込みフェイズで自陣のキャラクターアイディアゾーンが空 |
| 発動回数 | 1 試合 **1 回** だけ |
| 効果 | インスピレーションゾーン または 没案ゾーン から原案カード 1 枚を場に戻す |
| 没案からの場合のボーナス | 復活時に HP **100 ~ 50** 上乗せ |

---

# K. デッキ構築ルール

| 項目 | ルール |
|---|---|
| **メインデッキ枚数** | **60〜80 枚** 予定 |
| **デッキアイコン** | 1 枚必須(別カード扱い) |
| **どんでん返しデッキ** | 0〜5 枚(別山) |
| **同 ID カード上限** | 4 枚まで |
| **同 ID カードの場展開** | 自分の場に同 ID は **1 枚まで**(2 枚目以降は没) |

---

# L. カードの要素(共通)

| 要素 | 内容 |
|---|---|
| **コスト** | カードプレイに必要な IP(インスピレーションパワー) |
| **没で得る IP** | カード左上に 1〜3 で表記、没案ゾーン送り時に獲得 |
| **技**(キャラ) | 1〜2 個、IPコスト払って発動 |
| **HP**(キャラ) | 受けた技ダメージ合計を超えると没 |
| **イチオシ**(キャラ) | フラグ、HPが多く、没で相手原稿 +2 |

---

# M. 旧ルールとの差分(変更点まとめ)

| 旧ルール(v3 / 第一稿) | 新ルール(第二稿) |
|---|---|
| 主人公カード Lv1/2/3 進化 | **デッキアイコンカード**(進化なし、両面共通+裏面効果) |
| 主人公スキル(コスト払って技発動) | **両面共通効果**(常在) + **裏面効果**(覚醒時のみ) |
| 場 = キャラクターアイディアゾーン **6 枠** | **3 枠** に縮小 |
| バトルで没 → 没案ゾーン → IP獲得 | バトルで没 → **インスピレーションゾーン**、インスピ→没案 で IP 獲得 |
| 倒したキャラ自身が原稿になる | **デッキトップ** から 1〜2 枚を原稿ゾーンへ |
| 手札クリンナップは任意 | 就寝フェイズで **手札上限 5 枚** まで強制 |
| マリガン無制限ペナルティなし | **回数ペナルティ**(1 回ノーペナ、2 回目から戻し枚数増) |
| 自信作 (shinsaku) | **イチオシ** に改名 |
| どんでん返しは効果カードの一種 | **別山**(0〜5 枚)、アイコン覚醒時のみ使用 |
| 「サブ区分」が相棒/ライバル | **原案/ネーム/清書**(Lv に対応) |
