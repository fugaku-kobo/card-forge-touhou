# 東方カードゲーム ルール仕様(v2 ドラフト・覚醒モデル)

最終更新: 2026-05-19
状態: **ドラフト** ─ v1 (進化モデル) から **覚醒モデル** に切り替え済み。一部パラメータは未定(§J 参照)

このドキュメントは現行 東方カードゲームの **確定ルール** を一覧化したもの。
元データ: `詳細、ゲーム性.pdf`(2024-07) + `東方カードゲーム (1).xlsx`(2026-05)+ 仕様ヒアリング

> **用語ルール**: 「没」は使わず **「トラッシュ」** に統一する。
> **覚醒モデル**: 1 枚のカードに **通常時 / 覚醒時** の 2 セットのステータスを持たせ、コスト払って裏返す方式(同名カード重ねの「進化」は採用しない)。

---

# A. 勝敗条件

| 項目 | ルール |
|---|---|
| **勝利条件** | 相手の **ライフを 0** にする |
| **被弾(ライフ −1)** | 自デッキが 0 になった瞬間、デッキリフレッシュ処理(§G.4)が走り、結果としてライフが 1 減る |
| **引き分け** | 存在しない |
| **試合時間目安** | 10〜15 分 |

---

# B. デッキ構成

| 項目 | ルール |
|---|---|
| **メインデッキ枚数** | **50 枚** |
| 内訳目安 | スペル 20 枚 / キャラ 30 枚(仮) |
| **同 ID 上限** | **未定**(手塚版は 4 枚)|

> 旧設計の「進化ゾーン」「進化元別カード」は **廃止**。各キャラが 1 枚で通常↔覚醒の 2 状態を持つ。

スターターデッキ 2 種を発売予定、ブースターでカスタマイズ。

---

# C. カードタイプ(2 種類)

## C.1. キャラカード(覚醒モデル)

| 項目 | 内容 |
|---|---|
| **配置** | 盤面ゾーンに **1 枚のみ** 出せる |
| **レベル** | 1〜3(任意、スペルの発動条件用) |
| **攻撃力(通常時)** | バトル時に「攻撃力 − 相手回避力」分だけ相手デッキをトラッシュへ送る |
| **回避力(通常時)** | 相手の攻撃力から差し引く |
| **効果文(通常時)** | 登場時・常時効果(任意) |
| **覚醒コスト** | 覚醒(裏返し)に必要な P コスト |
| **覚醒時 攻撃力 / 回避力** | 覚醒後はこちらの値に切り替わる |
| **覚醒時 能力** | 覚醒時に発動する効果(例: 「1ドロー」) |

### 覚醒の挙動
- 任意のタイミング(進化フェイズ想定)で **覚醒コストを支払って裏返す**
- 裏返した後は **覚醒時** のステータスと能力が有効
- 一度覚醒すると **元に戻らない** 想定(要確認)

## C.2. スペルカード

| 項目 | 内容 |
|---|---|
| **発動条件 Lv**(左上) | 盤面キャラがこの Lv 以上なら使用可 |
| **種類** | `NORMAL` / `QUICK`(速攻、バトル中割り込み可)/ `COUNTER` 等 ─ 役割を区別 |
| **PP コスト** | P ゾーンから消費する量 |
| **BOM コスト** | BOMB ゾーンから消費する量(PP と **両建て** 可) |
| **トリガー** | 発動条件・タイミング(例: 「相手バトルフェイズ開始時」) |
| **回収 P 値**(右下、青) | 攻撃でトラッシュ送りされた時、攻撃側がこの値分トラッシュ→P ゾーンに置ける |
| **効果文** | カード固有効果 |

# D. ゾーン構造(8 ゾーン)

| ゾーン | 役割 |
|---|---|
| **デッキ** | 50 枚、攻撃で削られる(HP も兼ねる戦場) |
| **手札** | 初期 4 枚 |
| **盤面** | キャラ 1 体のみ(通常↔覚醒の状態を持つ) |
| **トラッシュ** | 旧称「没」。送られたスペルは攻撃側に P を還元 |
| **P ゾーン** | スペル発動・キャラ覚醒のコスト源 |
| **ライフ** | 3 枚裏向き、0 で敗北 |
| **BOMB ゾーン** | 3 枚裏向き、固定。スペルの BOM コスト or 2 枚消費で攻撃 1 回無効化 |
| **(進化ゾーンは廃止)** | ─ |

# E. ターン構造(6 フェイズ)

| # | フェイズ | 内容 |
|---|---|---|
| 1 | **ドロー** | デッキから 1 枚ドロー |
| 2 | **チャージ** | 手札 1 枚を P ゾーンに置き、1 ドロー(任意) |
| 3 | **覚醒** | キャラの覚醒処理(覚醒コストを支払って裏返し)。**先攻 1 ターン目はスキップ** |
| 4 | **メイン** | キャラ展開・スペル発動・伏せ等 |
| 5 | **バトル** | 攻撃宣言(§G) |
| 6 | **エンド** | クリンナップ・ターン終了 |

> v1 では「3. 進化」だったが、覚醒モデルへ切り替えに伴いフェイズ名を **「3. 覚醒」** に変更。

# F. セットアップ

| 手順 | 内容 |
|---|---|
| 1 | 互いにデッキシャッフル |
| 2 | **ライフ 3 枚** を裏向きで配置(デッキとは別に取り分け) |
| 3 | **BOMB 3 枚** を裏向きで配置(固定数) |
| 4 | **手札 4 枚** ドロー |
| 5 | 盤面にキャラ 1 枚を **伏せて** 設置 |

## マリガン

- 初手にキャラカードがない場合 → 手札公開 → 全戻し → シャッフル → 4 枚引き直し
- 回数制限なし・ペナルティなし

# G. バトル処理

## G.1. 攻撃宣言

自キャラ(通常 or 覚醒)で攻撃宣言する。
**ダメージ量 = max(0, 自攻撃力 − 相手回避力)**(値分だけ相手デッキをトラッシュへ送る)。

> 「キャラが没」という概念は存在しない。攻撃はあくまで **相手デッキを削る** 行為。
> ライフは原稿のように積み上げるのではなく、デッキ枯渇のペナルティとして 1 ずつ減る。

## G.2. 優先権(フラッシュタイミング)

バトル中はバトスピ風のフラッシュタイミング:

1. 攻撃側に優先権 → スペル発動可(1 枚解決)
2. 解決後、防御側に優先権 → スペル発動可(1 枚解決)
3. 両者連続放棄でバトル処理確定

(遊戯王のチェーン処理ではない、1 枚ずつ解決して優先権が動く方式)

`QUICK` 種別のスペルは特にバトル中の割り込みに使う想定。

## G.3. トラッシュ送り時の処理

相手デッキから送られたカードを順に確認:

- **スペル** が混ざっていれば、右下 **回収 P** の合計分、攻撃側はトラッシュから P ゾーンへ置ける

> v1 にあった「キャラが Lv +1 で進化して被弾しながら反撃」のルールは、**覚醒モデルでは想定しない**(同名重ね進化が無いため)。

## G.4. デッキリフレッシュ(被弾)

デッキが 0 枚になった瞬間:

1. トラッシュをすべてシャッフルしてデッキに戻す
2. 新デッキの **上 3 枚を新しいライフ** にする
3. 元のライフは…(処理未確定、§J)

### 分割吸収ルール
デッキが残り 5 枚しかない時に 20 枚分の攻撃を受けた場合:
- まず 5 枚をトラッシュへ送る
- リフレッシュ処理を行う
- 残り 15 枚分を新デッキから送る

ライフ 0 で敗北。

## G.5. BOMB の使い方

- スペルの BOM コストとして消費可能(PP コストと両建ても可)
- **BOMB 2 枚消費** で相手の攻撃を **1 回無効化**(=ダメージ 0)
- BOMB は 3 枚固定、補充手段なし(=実質 1 試合 1 回の防御札)

---

# H. 覚醒(キャラの状態切替)

## H.1. 覚醒条件

- 覚醒フェイズに、対象キャラの **覚醒コスト** を P ゾーンから支払う
- 支払うと **裏返して覚醒状態** に切り替え、以降は覚醒時のステータス・能力が有効
- ターン 1 回まで(要確認)
- **先攻 1 ターン目は覚醒不可**

## H.2. 覚醒時能力の発動

- 覚醒した瞬間、`覚醒時能力` が発動(登場時効果のような扱い)
- 例: 霊夢「1ドロー」、魔理沙「1ドロー」

## H.3. 元に戻るか

- 一度覚醒したら元に戻らない想定(要確認)

# K. 旧手塚版との差分(参考)

詳細は `docs/legacy/rule_status.md` 参照。要点だけ:

| 項目 | 手塚版 | 東方版 |
|---|---|---|
| 勝利条件 | 原稿 6 枚 | ライフ 0 |
| デッキ | 60〜80 枚 | 50 枚 |
| 場の上限 | 3 体 | **1 体** |
| 初期手札 | 7 枚 | 4 枚 |
| バトル | HP 削り合い、キャラが「没」 | **デッキ削り合い**、キャラはトラッシュにならない |
| マナ | IP(1 種類) | P と BOMB(2 種類、両建て可) |
| 罠カード | テコ入れ(伏せ) | (QUICK スペル等で割り込み) |
| 別山 | どんでん返し(0〜5 枚) | なし |
| キャラ強化 | Lv 1→2→3 の別カード重ね「進化」 | 1 枚で通常↔覚醒の **覚醒切替** |
| 専用カード | 主人公/デッキアイコン | なし |
| 特殊 | 自信作(イチオシ) | なし |
| フェイズ | ⓞ〜⑤ の漫画制作モチーフ | 1〜6 の TCG 一般用語(3. 覚醒) |
