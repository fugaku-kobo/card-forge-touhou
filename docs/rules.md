# 東方カードゲーム ルール仕様(v3.1 確定ドラフト・2リーダー制)

最終更新: 2026-05-22
状態: **確定ドラフト** ─ v2(覚醒モデル)に「2リーダー制」を導入し、`§J` の未確定事項を解消した版。残る項目はバランス調整のみ(§J)。

このドキュメントは現行 東方カードゲームの **確定ルール** を一覧化したもの。
元データ: `東方カードゲーム詳細、ゲーム性.pdf`(2024-07・原案)+ `東方カードゲーム (1).xlsx`(2026-05)+ 仕様ヒアリング + `card-forge`(手塚版)の 12 TCG 研究。

> **用語ルール**: 「没」は使わず **「トラッシュ」** に統一する。
> **覚醒モデル**: 1 枚のカードに **通常時 / 覚醒時** の 2 セットのステータスを持たせ、コストを払って裏返す方式。
> **2リーダー制(v3.1 新規)**: キャラカードは **デッキに入れず**、試合前に **2 体** 選んで場に固定する。デッキ 80 枚は **スペルカードのみ**。2 体は試合中ずっと表向き。攻撃・防御のたびに「どちらのリーダーを使うか」を選ぶのが読み合いの核(§G.1)。

---

# A. 勝敗条件

| 項目 | ルール |
|---|---|
| **勝利条件** | 相手の **ライフを 0** にする |
| **被弾(ライフ −1)** | 自デッキが 0 になった瞬間(**原因を問わない**)、デッキリフレッシュ処理(§G.4)が走り、ライフが 1 減る |
| **敗北** | 自分のライフが **0 になった瞬間** に敗北(=デッキを 3 回切らされたら負け) |
| **引き分け** | 存在しない(大会で時間切れの場合のみ、ライフの多い方を勝者とする) |
| **試合時間目安** | 約 15〜20 分 / 13〜20 ターン(デッキ 80・プレイ内容で変動) |

---

# B. リーダーとデッキ構成

| 項目 | ルール |
|---|---|
| **リーダー(キャラカード)** | 試合前に **2 体** 選び、リーダーゾーンに表向きで固定(試合中ずっと表)。デッキには **含めない** |
| **メインデッキ枚数** | **80 枚**(すべてスペルカード) |
| **同 ID 上限** | **5 枚**(`unlimited` フラグ付きカードは除外) |
| **構築の最低種類数** | スペル 16 種以上(80 ÷ 5 = 16) |

- スターターデッキ 2 種を発売予定。各スターターにリーダー複数体(最低 2 体)+ スペル 80 枚を同梱し、プレイヤーはそこから 2 体を選ぶ。
- ブースターパックでスペルを差し替えてカスタマイズする。
- 旧設計の「進化ゾーン」「キャラをデッキに入れる」「シューティングカード」は **すべて廃止**。

> **なぜ 2 リーダー制か**: 覚醒モデルでは「覚醒」に別カードが不要。旧 v2 のように盤面 1 体なのにキャラ 30 枚をデッキに入れると 29 枚が死に札になる。キャラをデッキ外に出してデッキ構築をスペルに一本化し、さらにリーダーを 2 体並べて「毎ターンどちらで攻める/受けるか」を選ばせることで、1 体制では枯れがちな中盤の読み合いを生む。

# C. カードタイプ(2 種類)

## C.1. キャラカード(=リーダー・覚醒モデル)

デッキには入れず、試合前に 2 体選んでリーダーゾーンに固定する本体カード。

| 項目 | 内容 |
|---|---|
| **配置** | リーダーゾーンに **2 体**(ともに表向き、試合中ずっと公開) |
| **通常時 Lv** | スペルの発動条件に参照(既定 **1**) |
| **攻撃力(通常時)** | バトル時に「攻撃力 − 相手回避力」分だけ相手デッキを削る |
| **回避力(通常時)** | 相手の攻撃力から差し引く |
| **効果文(通常時)** | 常時効果・登場時効果(任意) |
| **覚醒コスト** | 覚醒(裏返し)に必要な P コスト |
| **覚醒時 Lv** | 覚醒後に切り替わる Lv(通常時 Lv 以上。既定 **2**) |
| **覚醒時 攻撃力 / 回避力** | 覚醒後はこちらの値に切り替わる |
| **覚醒時 能力** | 覚醒した瞬間に発動する効果(例: 「1ドロー」) |

### リーダーの基本挙動
- リーダーは **トラッシュに送られない・破壊されない**(攻撃は相手デッキを削る行為であり、リーダーを倒す概念はない)。
- リーダー 2 体は試合を通して **同じ顔ぶれ**。途中で入れ替え・追加・除去はできない。
- 攻撃・防御のたびに 2 体のうちどちらを使うかを選ぶ(§G.1)。
- リーダーで攻撃すると、そのリーダーは **行動済み**(カードを横向きにする)になり、次の自分のターンまで **迎撃(防御)に使えない**。自分のターン開始時に通常向きへ戻る。
- 2 体は **別々に覚醒** できる(§H)。

## C.2. スペルカード

デッキ 80 枚を構成する唯一のカードタイプ。

| 項目 | 内容 |
|---|---|
| **発動条件 Lv**(左上) | 自分の **どちらかのリーダー** の現在 Lv がこの値以上なら使用可。`1`=常時 / `2`=いずれか覚醒後 / `3`=覚醒時 Lv3 のリーダーがいる時 |
| **PP コスト**(左下) | P ゾーンから消費する量 |
| **BOM コスト**(左下) | BOMB ゾーンから消費する量 |
| **トリガー** | 発動できるタイミング(例: 「メインフェイズ」「相手バトルフェイズ」)。`QUICK` 指定のものはバトル中の割り込みに使える |
| **回収 P 値**(右下・青) | このカードが攻撃で相手にトラッシュ送りされた時、攻撃した側がこの値分 P を得る(§G.3) |
| **効果文** | カード固有効果 |

> **`QUICK` について**: スペルは既定で「自分のメインフェイズのみ」使用可。`QUICK` を持つスペルだけが、相手ターンを含むバトル中の割り込み(フラッシュタイミング、§G.2)で使える。旧 v2 の `NORMAL`/`COUNTER` 等の細分類は廃止し、**通常 / QUICK の 2 種**に整理する。
>
> **`回収 P` の設計注意**: デッキが全スペルになったため、回収 P を全カードに付けると攻撃側に P が過剰供給される。**デッキの 3〜4 割程度のカードにのみ**設定すること。回収 P が高いカードは「強いが、削られると相手を利する」諸刃のカードとして価格付けする。

# D. ゾーン構造(7 ゾーン)

| ゾーン | 役割 |
|---|---|
| **デッキ** | 80 枚。攻撃で削られる戦場(=実質ライフ) |
| **手札** | 初期 4 枚(後攻は 5 枚)。上限 7 枚 |
| **リーダーゾーン** | キャラ 2 体(各々が通常↔覚醒の状態を持つ)。デッキ外 |
| **トラッシュ** | 旧称「没」。削られたカードが置かれる |
| **P ゾーン** | スペル発動・リーダー覚醒のコスト源 |
| **ライフ** | 3 枚、裏向き。デッキリフレッシュごとに 1 枚ずつ手札へ。0 枚で敗北 |
| **BOMB ゾーン** | 3 枚、裏向き、固定。スペルの BOM コスト、または 2 枚消費で攻撃 1 回を無効化 |

> 旧 v2 の「進化ゾーン」「盤面ゾーン」は廃止(盤面 → リーダーゾーンに統合)。

# E. ターン構造(6 フェイズ)

| # | フェイズ | 内容 |
|---|---|---|
| 1 | **ドロー** | 行動済みのリーダーを通常向きに戻し、デッキから 1 枚ドロー |
| 2 | **チャージ** | 手札 1 枚を P ゾーンに置き、1 枚ドロー(任意・ターン 1 回) |
| 3 | **覚醒** | リーダーの覚醒処理(覚醒コストを払って裏返す)。1 ターンに覚醒できるのは **2 体のうち 1 体まで** |
| 4 | **メイン** | スペルの発動・伏せ等 |
| 5 | **バトル** | 攻撃宣言(§G) |
| 6 | **エンド** | クリンナップ。手札が **7 枚** を超えていたら超過分を捨てる。ターン終了 |

## 最初のターンの制限

- **どちらのプレイヤーも**、自分の最初のターンは **覚醒不可**(攻撃は可能)。
- **先攻プレイヤー**は、加えて最初のターンの **ドローフェイズをスキップ**(先攻有利の緩和)。

> チャージ → 覚醒 → メイン → バトルの順なので、2 ターン目以降は「覚醒して、その上昇したステータスで同じターンに攻撃する」テンポプレイが可能。

# F. セットアップ

| 手順 | 内容 |
|---|---|
| 1 | 互いにリーダー(キャラカード)を **2 体** 選び、リーダーゾーンに **表向き**(通常時)で置く |
| 2 | デッキ 80 枚をシャッフル |
| 3 | **ライフ 3 枚** をデッキトップから取り、裏向きで取り分ける |
| 4 | **BOMB 3 枚** をデッキトップから取り、裏向きで配置(固定数) |
| 5 | **手札ドロー** ── 先攻 **4 枚** / 後攻 **5 枚**。さらに **後攻は P ゾーンに P を 6 個** 持って開始(後攻補正 ── 先攻有利の緩和) |
| 6 | マリガン(下記) |
| 7 | 先攻・後攻をランダムに決定 |

## マリガン

- 初手が気に入らなければ、**1 回だけ** 手札を全部デッキに戻してシャッフルし、同じ枚数を引き直せる。
- ペナルティなし。

> v2 の「初手にキャラカードがなければ引き直し」ルールは廃止(キャラはデッキに入らないため不要)。

# G. バトル処理

## G.1. 攻撃宣言(2 リーダーの読み合い)

バトルフェイズに攻撃宣言する。攻撃は **ターン 1 回**。

1. **攻撃側** は自分のリーダー 2 体から **1 体を選んで攻撃** する(=そのリーダーの「技」を使う)。攻撃したリーダーは **行動済み**(横向き)になり、次の自分のターンまで迎撃に使えない。
2. **防御側** は **行動済みでない** 自分のリーダー 1 体で迎え撃つ(前ターンに攻撃していれば、残った 1 体が自動的に迎撃役になる)。その回避力を参照。
3. **削り量(ダメージ) = max(1, 攻撃リーダーの攻撃力 − 迎撃リーダーの回避力)**
4. その値の分だけ防御側のデッキの上からトラッシュへ送る。
5. 攻撃に使ったリーダーが「攻撃時」に誘発する能力を持つ場合は、それも発動する。

> **読み合いの核 ─「攻撃役を選ぶ＝次の防御役を決める」**
> 攻撃に使ったリーダーは次の自分のターンまで行動済みなので、迎撃に回れない。
> つまり「今ターンどちらで攻撃するか」の選択が、そのまま「次の相手ターンにどちらで受けるか」を決める。1 つの選択が攻防の両方にかかるのが、このゲームの読み合いの中心。
> 例: 火力の高いリーダーで殴れば大きく削れるが、そのリーダーは次の相手ターンに受けへ回れず、もう 1 体で耐えることになる。
> 2 体のステータスは常に公開。隠れた情報は **手札のスペル**(どちらのリーダーに攻撃ブーストを乗せるか等)だけ。
>
> **最低 1 保証**: 迎撃リーダーの回避力が攻撃力以上でも、攻撃は **必ず 1 枚以上** 削る。完全な膠着が起きない。
> 攻撃は相手デッキを削る行為であり、「リーダーが没になる」概念は存在しない。

## G.2. 優先権(フラッシュタイミング)

バトル中はバトスピ風のフラッシュタイミングで `QUICK` スペルを使える:

1. 攻撃側に優先権 → `QUICK` スペルを発動可(1 枚解決)
2. 解決後、防御側に優先権 → `QUICK` スペルを発動可(1 枚解決)
3. 両者が連続で優先権を放棄したらバトル処理(削り)を確定

(遊戯王のチェーン処理ではない。1 枚ずつ解決して優先権が動く方式)

## G.3. トラッシュ送り時の処理(回収 P)

攻撃で相手デッキから送られたカードを確認し、**回収 P 値の合計分**(ただし **1 回の攻撃につき最大 3**)、攻撃した側がトラッシュからカードを P ゾーンへ置ける。

> **上限 3 の理由**: キャップがないと、1 回の攻撃で攻撃側に十数 P が一気に入り、雪だるま式に有利が加速する(v1 プレイテスト #1 で判明)。1 攻撃あたり最大 3 P に制限する。

## G.4. デッキリフレッシュ(被弾)

**自デッキが 0 枚になった瞬間**(相手の攻撃・自分の効果による自傷・ドロー切れ・直接削りなど、**原因を問わない**):

1. トラッシュをすべてシャッフルしてデッキに戻す
2. **自分のライフを 1 枚、裏向きのまま手札に加える**(被弾の救済)
3. ライフが **0 になったら、その瞬間に敗北**

> 旧 v2 §G.4 の「新デッキの上 3 枚を新しいライフにする」は原案からの **転記ミス**。正しくは上記の通り、ライフは試合開始時に取り分けた 3 枚を使い、リフレッシュごとに 1 枚ずつ手札に移す(ONE PIECE / 開運コロシアムの「ライフ→手札 救済」と同型)。

### 分割吸収ルール

デッキが残り 5 枚の時に 20 枚分の攻撃を受けた場合:

1. まず 5 枚をトラッシュへ送る
2. リフレッシュ処理(上記 1〜3)を行う
3. 残り 15 枚分を、戻したデッキから続けてトラッシュへ送る

## G.5. BOMB の使い方

- スペルの **BOM コスト**として消費できる。
- **BOMB 2 枚消費**で、相手の攻撃を **1 回無効化**(=削り 0)。
- BOMB は 3 枚固定・補充手段なし(=実質 1 試合 1〜2 回の防御札)。

# H. 覚醒(リーダーの状態切替)

## H.1. 覚醒条件

- 覚醒フェイズに、覚醒させたいリーダーの **覚醒コスト** を P ゾーンから支払う。
- 支払うとそのリーダーを **裏返して覚醒状態** に切り替え、以降は覚醒時のステータス(Lv・攻撃力・回避力)が有効になる。
- 2 体のリーダーは **別々に覚醒** できるが、**1 ターンに覚醒できるのは 1 体まで**(両方覚醒させるには最低 2 ターンかかる)。
- **自分の最初のターンは覚醒不可**(§E)。

## H.2. 覚醒時能力の発動

- 覚醒した瞬間、`覚醒時能力` が発動する(登場時効果のような扱い)。
- 例: 霊夢「1ドロー」、魔理沙「1ドロー」。

## H.3. 覚醒の不可逆性

- 一度覚醒したら **元には戻らない**。
- 覚醒で Lv が上がると、それまで使えなかった高 Lv スペル(発動条件 Lv 2・3)が解禁される。**覚醒は中盤の明確なパワースパイク**として機能する。

# I. 各種数値・バランス指針

## I.1. 確定値

| 項目 | 値 |
|---|---|
| デッキ枚数 | 80(スペル) |
| リーダー | 2(デッキ外) |
| 手札初期 | 先攻 4 / 後攻 5(上限 7) |
| ライフ | 3 |
| BOMB | 3(固定) |
| 同 ID 上限 | 5 |
| 試合時間 | 約 15〜20 分 |

## I.2. バランス指針(カード設計の目安)

> **最重要パラメータ**: 原案では「弾幕連打のように **1 撃で 10〜30** 削る」のが想定。素の攻撃力だけでは届かず、**スペルで攻撃力を上げながら殴る**のがゲームの核。カード設計はこの数値感に合わせる。

| 指標 | 目安 |
|---|---|
| 基礎攻撃力(リーダー) | 12〜25 |
| 基礎回避力(リーダー) | 5〜12 |
| 素の削り量(攻撃力 − 回避力) | 5〜15 |
| スペル補正込みの 1 撃 | 15〜40(弾幕らしい大味) |
| 覚醒時の攻撃力上昇 | +5〜10 |
| デッキ 1 本(80 枚)を削り切る攻撃回数 | 3〜5 回 |

> 現行プロトカード(霊夢 攻13/回避5、魔理沙 攻8/回避10)は素の削り量が 3 前後で **低すぎる**。上記の指針に沿って再調整が必要(§J)。

| 先攻補正 | 覚醒スキップのみ | ドローもスキップ + 覚醒不可 |
| マリガン | 初手にキャラ無→引き直し | 1 回まで全戻し 4 枚(キャラ条件は廃止) |

# L. 旧手塚版との差分(参考)

詳細は `docs/legacy/rule_status.md` 参照。要点:
